# Introdução

## O que é o `filter()`?

O método `filter()` é uma função de ordem superior disponível para **arrays em JavaScript**;

Ele é usado para **criar um novo array contendo apenas os elementos que passam em um teste (condição) especificado em uma função callback**.

Em resumo:

> `filter()` **filtra os elementos de um array** com **base em uma condição** que retorna `true` ou `false`.

---
# Sintaxe

```js
array.filter(function(elemento, indice, arrayOriginal){
	// retorna true para manter o elemento
	// retorna false para descartá-lo
}, thisArg)
```

### Parâmetros

- `elemento` → o item atual do array que está sendo processado.
- `indice` *(opcional)* → o índice do item atual.
- `arrayOriginal` *(opcional)* → o array completo que está sendo percorrido.
- `thisArg` *(opcional)* → valor a ser usado como `this` dentro da função callback.

### Retorno

- Um **novo array contendo apenas os elementos que passaram no teste.**

---
# Como Funciona Passo a Passo

1. O `filter()` percorre todos os elementos do array original.
2. Para cada elemento, ele executa a função callback.
3. Se a função callback retornar `true`, o elemento é incluído no novo array.
4. Se retornar `false`, o elemento é descartado.
5. O array original não é modificado.

---
# Exemplo Simples

```js
const numeros = [1, 2, 3, 4, 5, 6];

// Filtrar apenas os números pares
const pares = numeros.filter(numero => numeros % 2 === 0);

console.log(pares); // [2, 4, 6]
```

**Explicação:**

- Para cada número do array, verifica se `numero % 2 === 0`.
- Se for verdadeiro, o número entra no novo array `pares`.

---
# Exemplo com Objetos

```js
const alunos = [
	{ nome: "Ana", nota: 8 },
	{ nome: "João", nota: 5 },
	{ nome: "Maria", nota: 9 }
];

// Filtrar apenas os alunos aprovados (nota >= 7)
const aprovados = aunos.filter(aluno => aluno.nota >= 7);

console.log(aprovados);
// Saída: [ { nome: 'Ana', nota: 8 }, { nome: 'Maria', nota: 9 }]
```

---
# Pontos Importantes

- `filter()` **não altera o array original**.
- Pode ser combinado com outros métodos como `map()` e `reduce()` para processamento funcional de dados.
- Caso nenhum elemento passe no teste, o `filter()` retorna um array **vazio**.
- Se a função callback **nunca retornar `true`**, o array final terá **0 elementos**.

---
# Quando Usar

Use o `filter()` quando precisar **selecionar elementos específicos de um array** com base em uma condição, como:

- Filtrar produtos acimo de certo preço.
- Filtrar nomes que começam com determinada letra.
- Filtrar usuários ativos em uma lista de objetos.
