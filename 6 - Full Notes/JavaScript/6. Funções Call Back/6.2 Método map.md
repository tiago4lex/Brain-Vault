# Introdução

## O que é o `map()`?

O método `map()` é uma **função própria dos arrays** em JavaScript.

Ele serve para **percorrer todos os elementos de um array e aplicar uma função a cada um deles,** cirando um **novo array com valores transformados.**

>[!attention] Atenção:
>O `map()` **não altera o array origina** - ele criar e retorna um **novo arrya** com os resultados da transformação.

---
# Sintaxe

```js
array.map(function(elemento, indice, arrayOriginal){
	// Código para transformar cada elemento
});
```

## Parâmetros da função de callback

- `elemento` → o elemento atual do array.
- `índice` *(opcional)* → o índice do elemento atual.
- `arrayOriginal` *(opcional)* → o array inteiro sendo perecorrido.

## Valor de retorno

- Um **novo array** com os elementos transformados pela função callback.

---
# Exemplo básico

```js
const numeros = [1, 2, 3, 4, 5];

const dobrados = numeros.map(function(numro){
	return numero * 2;
});

console.log(dobrados);  // [2, 4, 6, 8, 10]
console.log(numeros);   // [1, 2, 3, 4, 5] (permanece igual)
```

### O que acontece:

1. `map()` percorre cada número do array `numeros`.
2. Para cada número, executa `numero * 2`.
3. Cria um novo array com os valores multiplicados por 2.
4. O array original não é modificado.

---
# Exemplo com Arrow Function (forma mais moderna).

```js
const numeros [1, 2, 3, 4, 5];

const dobrados = numeros.map(n => n * 2);

console.log(dobrados); // [2, 4, 6, 8, 10]
```

# Exemplo usando índice

```js
const letras = ['a', 'b', 'c'];

const resultado = letras.map((letram, indice) => {
	return `${undice}: ${letra}`;
});

console.log(resultado);  // ["0: a", "1: b", "2: c"]
```

# Exemplo com objetos

```js
const produtos = [
    { nome: 'Notebook', preco: 3500 },
    { nome: 'Mouse', preco: 80 },
    { nome: 'Teclado', preco: 150 }
];

const nomes = produtos.map(item => item.nome);

console.log(nomes); // ["Notebook", "Mouse", "Teclado"]
```

---
# Características importantes

- `map()` **sempre retorna um novo array**.
- O tamanho do novo array será **o mesmo do original**.
- **Não modifica** o array original.
- É ideal para **transformar dados** (não serve para filtrar ou somar — para isso existem `filter()` e `reduce()`).

---
# Diferença entre `map()` e `forEach()`

| Característica        | `map()`           | `forEach()`                 |
| --------------------- | ----------------- | --------------------------- |
| Retorna um novo array | ✅ Sim             | ❌ Não (retorna `undefined`) |
| Altera o original     | ❌ Não             | ❌ Não                       |
| Uso ideal             | Transformar dados | Executar efeitos colaterais |
