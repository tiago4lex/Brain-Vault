# Introdução

## O que é o Operador Spread

O **operador spread (`...`)** é um recurso introduzido no JavaScript com o ECMAScript 6 (ES6).

Ele serve para **"espalhar" (expandir)** os elementos de um **array, objeto ou iterável** em outro local, como:

- dentro de novos arrays
- dentro de novos objetos
- como argumentos de funções

---
# Sintaxe Básica

```js
const novoArray = [...arrayExistente];
const novoObjeto = {...objetoExistente};
funcao(...valores);
```

---
# Usando Spread com Arrays

## Copiando um Array

```js
const numeros = [1, 2, 3];
const copia = [...numeros];

console.log(copia); // [1, 2, 3]
```

> A cópia é **independente** do array original.

## Concatenando Arrays

```js
const pares = [2, 4, 6];
const impares = [1, 3, 5];

const numeros = [...pares, ...impares];

console.log(numeros); // [2, 4, 6, 1, 3, 5]
```

## Adicionado elementos no meio

```js
const numeros = [1, 2, 3]
const novos = [0, ...numeros, 4];

console.log(novos); // [0, 1, 2, 3, 4]
```

---
# Usando Spread com Objetos

## Copiando um Objeto

```js
const dadosPessoais = { nome: "João", idade: 21 };
const endereco = { cidade: "São Paulo", estado: "SP"};

const completo = { ...dadosPessoais, ...endereco };

console.log(completo);
// { nome: "João", idade: 21, cidade: "São Paulo", estado: "SP" }
```

>[!attention] Atenção:
>Se houver **chaves iguais**, o último objeto sobrepõe os anteriores.

---
# Usando Spread em Funções

O spread permite passar **cada elemento de um array como argumento separado** para uma função.

```js
const numeros = [5, 10, 15];

function soma(a, b, c) {
	return a + b + c;
}

console.log(soma(...numeros)); // 30
```

Sem o spread, seria necessário escrever `soma(numeros[0], numeros[1], numeros[2])`.

---
# Diferença entre Spread e Rest

- **Spread (`...`):** **espalha** elementos de um array/objeto.
- **Rest (`...`):** **junta** vários argumentos em um array (usado na definição de funções).

```js
// Spread
const numeros = [1, 2, 3];
console.log(...numeros); // 1 2 3

// Rest
function soma(...valores) {
	return valores.reduce((a, b) => a + b, 0);
}

console.log(soma(1, 2, 3)); // 6
```

✅ Combinar vários arrays em um só  
✅ Combinar vários objetos em um só  
✅ Passar arrays como argumentos para funções

# Resumo

- O operador `...` **espalha elementos de arrays, objetos ou iteráveis**
- Ajuda a escrever código **mais limpo e enxuto**
- Muito usado junto com métodos imutáveis (como `map`, `filter`, `reduce`)
