# Introdução
## Oque são Operadores Lógicos?

Operadores Lógicos são símbolos usados para realizar **operações de lógica booleana**. Eles retornam **true** ou **false**, dependendo da avaliação das expressões.

Muito usados em **condições** (`if`, `while`, etc.) e em **comparação múltiplas**.

---
# Principais Operadores Lógicos

| Operador | **Nome**   | **Função**                                                  |
| -------- | ---------- | ----------------------------------------------------------- |
| `&&`     | E lógico   | Retorna **true** se ambas as condições forem verdadeiras    |
| `!`      | NÃO lógico | Inverte o valor lógico (true → false, false → true)         |
| \| \|    | OU lógico  | Retorna **true** se apenas uma das condições for verdadeira |

---
# Operador E lógico (`&&`)

## Funcionamento

- **true && true** → `true`
- **true && false** → `false`
- **false && true** → `false`
- **false && false** → `false`

## Exemplo

```js
let idade = 20;
let temCarteira = true;

if (idade >= 18 && temCarteira) {
	console.log("Pode dirigir");
} else {
	console.log("Não pode dirigir");
}
```

✅ Só imprime "Pode dirigir" se **ambas** as condições forem verdadeira.

---
# Operador OU lógico (`||`)

## Funcionamento

- **true || true** → `true`
- **true || false** → `true`
- **false || true** → `true`
- **false || false** → `false`

## Exemplo

```js
let ehFeriado = false;
let ehFinalSemana = true;

if (ehFeriado || ehFinalSemana) {
	console.log("Hoje é dia de descanso");
} else {
	console.log("Hoje é dia de trabalho");
}
```

---
# Operador NÃO lógico (`!`)

## Funcionamento

- **!true** → `false`
- **!false** → `true`

## Exemplo

```js
let chovendo = false;

if (!chovendo) {
	console.log("Pode sair para passear");
} else {
	console.log("Leve guarda-chuva");
}
```

✅ Inverte o valor lógico.

---
# Combinação de Operadores

Podemos combinar operadores para criar condições mais complexas.

```js
let idade = 25;
let temCNH = true;
let carroDisponivel = false;

if ((idade >= 18 && temCNH) || carroDisponivel) {
	console.log("Pode viajar de carro");
} else {
	console.log("Não pode viajar de carro");
}
```

---
# Curto-Circuito *(Short-Circuit Evaluation)*

Os operadores `&&` e `||` podem **retornar valores não booleanos** e parar a execução assim que o resultado final é definido.

## Com `&&`

```js
console.log(false && "Olá"); // false
console.log(true && "Olá");  // "Olá"
```

## Com `||`

```js
console.log(true || "Olá");  // true
console.log(false || "Olá"); // "Olá"
```

Esse comportamento é muito usado para definir valores padrões:

```js
let nome = "";
let usuario = nome || "Visitante";

console.log(usuario); // Visitante
```

# Tabela

## Operador `&&` (E lógico)

| A     | B     | A && B |
| ----- | ----- | ------ |
| true  | true  | true   |
| true  | false | false  |
| false | true  | false  |
| false | false | false  |

## Operador `||` (OU lógico)

| A     | B     | A \| B |
| ----- | ----- | ------ |
| true  | true  | true   |
| true  | false | true   |
| false | true  | true   |
| false | false | false  |

## Operador `!` (NÃO lógico)

| A     | !A    |
| ----- | ----- |
| true  | false |
| false | true  |

---
# Boas Práticas

✅ Use `&&` quando todas as condições precisam ser verdadeiras.  
✅ Use `||` quando pelo menos uma precisa ser verdadeira.  
✅ Use `!` para negar valores e simplificar comparações.  
✅ Sempre use parênteses para deixar a condição clara:

```js
if ((idade > 18 && temCNH) || ehMotoristaProfissional) { ... }
```
