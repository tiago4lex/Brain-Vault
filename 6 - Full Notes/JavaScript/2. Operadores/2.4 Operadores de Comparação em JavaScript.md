# Introdução

## O que são Operadores de Comparação?

Operadores de comparação são usados para **comparar dois valores** e retornar um resultado **booleano** (`true` ou `false`).

Eles são amplamente usados em **estruturas condicionais** (`if`, `while`, `switch`) e em **expressões lógicas**.

---
# Lista de Operadores de Comparação

| Operador | Nome                            | Descrição                                                            | Exemplo     | Resultado |
| -------- | ------------------------------- | -------------------------------------------------------------------- | ----------- | --------- |
| ==       | Igual a (comparação solta)      | Compara dois valores, **fazendo conversão de tipo** se necessário    | `5 == "5"`  | `true`    |
| ===      | Estritamente igual              | Compara dois valores **sem conversão de tipo**                       | `5 === "5"` | `false`   |
| `!=`     | Diferente de (comparação solta) | Verifica se os valores são diferentes, **fazendo conversão de tipo** | `5 != "5"`  | `false`   |
| `!==`    | Estritamente diferente          | Verifica se os valores e tipos são diferentes                        | `5 !== "5"` | `true`    |
| `>`      | Maior que                       | Verifica se o valor da esquerda é maior                              | `10 > 5`    | `true`    |
| `<`      | Menor que                       | Verifica se o valor da esquerda é menor                              | `10 < 5`    | `false`   |
| `>=`     | Maior ou igual a                | Verifica se o valor da esquerda é maior ou igual                     | `5 >= 5`    | `true`    |
| `<=`     | Menor ou igual a                | Verifica se o valor da esquerda é menor ou igual                     | `3 <= 5`    | `true`    |

---
# Diferença entre == e ===

- == **(comparação solta)**: converte automaticamente os tipos antes de comparar ***(type coercion)***.
- === **(comparação estrita)**: não converte tipos, os valores devem ser iguais e do mesmo tipo.

## Exemplo

```js
console.log(5 == "5");    // true → converte "5" para número antes de comparar
console.log(5 === "5");   // false → tipos diferentes (number vs string)
```

---
# Diferença entre `!=` e `!==`

- `!=`: converte tipos antes de comparar.
- `!==`: não converte tipos, precisa que tipos e valores sejam diferentes.

## Exemplo

```js
console.log(5 != "5");   // false → "5" é conveertido para número antes de comparar
console.log(5 !== "5"); // ture → tipos diferentes
```

---
# Comparação com Strings

Strings são comparadas **lexicograficamente** (pela ordem dos caracteres na tabela Unicode).

## Exemplo

```js
console.log("banana" > "abacaxi");  // true → "b" vem depois de "a"
console.log("Zebra" > "abelha");    // false → "Z" ten código menor que "a"
```

---
# Comparação com Tipos Diferentes

Quando usamos == , o JavaScript tenta **converter os valores para o mesmo tipo** antes de comparar. Com === , não há conversão.

## Exemplo

```js
console.log(true == 1);   // true → true é convertido para 1
console.log(true === 1);  // false → tipos diferentes (boolean vs number)

console.log(null == undefined)   // true → considerados iguais pelo ==
console.log(null === undefined)  // false → tipos diferentes
```
