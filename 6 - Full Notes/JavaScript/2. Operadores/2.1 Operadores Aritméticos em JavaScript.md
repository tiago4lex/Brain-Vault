# Introdução

## O que são Operadores Aritméticos

Os **operadores aritméticos** são símbolos usados para realizar **operações matemáticas** sobre valores ou variáveis numéricas.

---
# Principais Operadores Aritméticos

| **Operador** | **Nome**       | **Exemplo** | **Resultado** |
| ------------ | -------------- | ----------- | ------------- |
| `+`          | Adição         | `5 + 3`     | `8`           |
| `-`          | Subtração      | `5 - 3`     | `2`           |
| `*`          | Multiplicação  | `5 * 3`     | `15`          |
| `/`          | Divisão        | `10 / 2`    | `5`           |
| `%`          | Módulo (resto) | `10 % 3`    | `1`           |
| `**`         | Exponenciação  | `2 ** 3`    | `8`           |

---
# Exemplos de Uso

## Adição

```js
let a = 10;
let b = 5;

console.log(a + b); // 15
```

Também funciona para **concatenação** de strings:

```js
let nome = "Maria"
let saudacao = "Olá" + nome;

console.log(saudacao); // Olá Maria
```

## Subtração

```js
let x = 20;
let y = 7;

console.log(x - y); // 13
```

## Multiplicação

```js
let largura = 4;
let altura = 5;
let area = largura * altura;

console.log(area); // 20
```

## Divão

```js
let total = 100;
let partes = 4;

console.log(total / partes); // 25
```

## Módulo (resto)

```js
console.log(10 % 3); // 1
console.log(15 % 4); // 4
```

Muito usado para verificar **números pares ou ímpares**:

```js
let numero 7;

if numero % 2 === 0) {
	console.log("Par");
} else {
	console.log("Ímpar");
}
```

## Exponenciação

```js
console.log(2 ** 3); // 8
console.log(5 ** 2); // 25
```

---
# Operadores de Incremento e Decremento

## Incremento (`++`)

Adiciona `1` ao valor da variável.

```js
let n = 5;
n++;

console.log(n); // 6
```

## Decremento (`--`)

Subtrai `1` do valor da variável.

```js
let m = 5;
m--;

console.log(m); // 4
```

---
# Ordem de Precedência

A ordem em que os operadores são avaliados segue as regras matemáticas **(PEMDAS)**:

1. **Parênteses `()`**
2. **Exponenciação `**`**
3. **Multiplicação `*`, Divisão `/`, Módulo `%`**
4. **Adição `+` e Subtração `-`**

**Exemplo:**

```js
console.log(2 + 3 * 4);   // 14 (3 * 4 é feito antes)
console.log((2 + 3) * 4); // 20 (parênteses mudam a ordem)
```

---
# Cuidados Importantes

- Divisão por zero em JavaScript **não gera erro**, mas retorna `Infinity`:

```js
console.log(5 / 0); // Infinity
```

- Operações com valores inválidos resultam em `NaN` ***(Not a Number)***:

```js
console.log("abc" * 2) // NaN
```

- Operador `+` com strings pode causar **concatenação indesejada**:

```js
console.log(5 + "3"); // "53" (string)
```

Para evitar, converta antes:

```js
console.log(5 + Number("3")); // 8
```
