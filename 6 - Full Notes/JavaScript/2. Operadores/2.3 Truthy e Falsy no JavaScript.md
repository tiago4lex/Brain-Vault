# Introdução

## Oque são Truthy e Falsy?

Em JavaScript, **qualquer valor** pode ser avaliado como **verdadeiro ou falso** quando usado em um contexto que espera um valor booleano, como em condições (`if`, `while`, `?:`, etc.).

- **Truthy** → valores que o JavaScript considera como `true`.
- **Falsy** → valores que o JavaScript considera como `false`.

> Isso acontece por causa da **coerção de tipos** *(type coercion)*, onde o JavaScript converte automaticamente valores para booleanos em determinadas situações.

---
# Valores Falsy

São exatamente **7 valores** que sempre são considerados `false` em um contexto booleano:

| **Valor**   | **Tipo**  | **Observação**    |
| ----------- | --------- | ----------------- |
| `false`     | Boolean   | Literal falso     |
| `0`         | Number    | Zero inteiro      |
| `-0`        | Number    | Zero negativo     |
| `0n`        | BigInt    | Zero como BigInt  |
| `""`        | String    | String vazia      |
| `null`      | Object    | Ausência de valor |
| `undefined` | Undefined | Valor indefinido  |
| `NaN`       | Number    | Not-a-Number      |

## Exemplo

```js
if (0) {
	console.log("Truthy");
} else {
	console.log("Falsy")
}

// Saída: "Falsy"
```

---
# Valores Truthy

São **todos os outros valores** que não estão na lista Falsy.

Exemplos comuns de Truthy:
- Números diferentes de `0` (positivos ou negativos) → `42`, `-5`
- Strings não vazias → `"Olá"`, `"false"`
- Objetos e arrays (mesmo vazias) → `{}`, `[]`
- Funções → `function() {}`
- Valores especiais como `Infinity` e `-Infinity`

## Exemplo

```js
if ("JavaScript") {
	console.log("Truthy");
} else {
	console.log("Falsy");
}

// Saída: "Truthy"
```

---
# Testando Truthy e Falsy

Podemos converter um valor para booleano usando:

- **Função** `Boolean()`
- **Operador de negação dupla** `!!`

## Exemplo

```js
console.log(Boolean(""));      // false
console.log(Boolean("JS"));    // true

console.log(!!0);              // false
console.log(!!123);            // true
```

# Uso prático de Truthy e Falsy

## Condição mais simples

```js
let nome = "";

if (nome) {
	console.log("Nome informado:", nome);
} else {
	console.log("Nome não informado");
}
```

## Valores padrão com `||`

```js
let usuario = "";
let nomeExibicao = usuario || "Visitante";

console.log(nomeExibicao);  // "Visitante"
```

## Checagem de propriedades de objetos

```js
let config = { tema: "escuro" };

if config.tema {
	console.log("Tema definido:", config.tema);
} else {
	console.log("Tema não definido");
}
```

---
# Cuidado com armadilhas

Nem sempre um valor ***aparentemente verdadeiro*** é Truthy:


```js
console.log(Boolean("0"));     // true (string "0" não é vazia)
console.log(Boolean([]));      // true (array vazia é Truthy)
console.log(Boolean({}));      // true (objeto vazio é Truthy)
console.log(Boolean(NaN));     // false
```
