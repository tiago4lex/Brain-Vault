# Introdução

## O que é a procura em listas?

Quando trabalhamos com arrays, muitas vezes é necessário **verificar se um item existe** e, se existir, **descobrir sua posição** para recuperar informações associadas a ele.

No exemplo a seguir, temos duas listas relacionadas:

- Uma lista de alunos.
- Uma lista de médias (notas).

Cada aluna está vinculado à sua média pelo **índice** (posição no array).

```js
const alunos = ["João", "Julina", "Caio", "Ana"];
const medias = [10, 8, 7.5, 9];

const lista = [alunos, medias];
```

Aqui:

- `alunos[0]` → `João`
- `media[0]` → 10

Ou seja, **João tem média 10**.

---
# Função de busca

A função de exemplo tem três coisas principais:

```js
function exibeNomeENota(aluno) {
	if(lista[0].includes(aluno)) {    // 1. Verifica se o aluno existe
		const indice = lista[0].indexOf(aluno);    // 2. Descobre a posição do aluno
		const mediaAluno = lista[1][indice];    // 3. Usa o índice para pegar a nota
		console.log(`${aluno} tem a média ${mediaAluno}`);
	} else {
		console.log("Estudante não existe na lista!");
	}
}
```

## 1. Explicando os métodos usados

- `includes()`
	- Verifica se um valor existe no array.
	- Retorna `true` ou `false`.

```js
["a", "b", "c"].includes("b"); // true
["a", "b", "c"].includes("z"); // false
```

- `indexOf()`
	- Retorna o índice (posição) do elemento.
	- Se não existir, retorna `-1`.

```js
["a", "b", "c"].indexOf("c"); // 2
["a", "b", "c"].indexOf("z"); // -1
```

---
# Usando a função do exemplo

```js
exibeNomeENota("Juliana"); // Juliana tem a média 8
exibeNomeENota("Maria");   // Estudante não existe na lista!
```

# Outras formas de procurar em arrays

## 1. Usando `find()` para objetos

Se tivermos alunos e notas em **objetos**, podemos usar `find()`.

```js
const alunos = [
	{ nome: "João", media: 10 },
	{ nome: "Juliana", media: 8 },
	{ nome: "Caio", media: 7.5 },
	{ nome: "Ana", media: 9 }
];

function exibeNomeENota(aluno) {
	const encontrado = alunos.find(a => a.nome === aluno);
	if(encontrado) {
		console.log(`${encontrado.nome} tem a média ${encontrado.media}`);
	} else {
		console.log("Estudante não existe na lista!");
	}
}

exibeNomeENota("Caio");   // Caio tem a média 7.5
exibeNomeENota("Maria");  // Estudante não existe na lista!
```

> **Vantagem:** Código mais legível e estruturado.

### `a => a.nome === aluno`:

Esse trecho é uma ***arrow function***. É uma forma mais curta e moderna de escrever funções.

- Forma tradicional (antes do ES6)

```js
function verificar(a) {
	return a.nome === aluno;
}
```

- Forma com *Arrow Function*

```js
(a) => a.nome === aluno;
```

As duas versões fazem a **mesma coisa**: recebem um parâmetro chamado `a` e retornam o resultado da comparação `a.nome === aluno`.

#### O que é o `a`?

O `a` é apenas o **nome do parâmetro**.

Ele representa **cada elemento** do array enquanto o método (`find`, `filter`, etc.) percorre os valores.

Exemplo:

```js
const alunos = [
  { nome: "João", media: 10 },
  { nome: "Juliana", media: 8 },
  { nome: "Caio", media: 7.5 },
  { nome: "Ana", media: 9 }
];

alunos.find(a => a.nome === "Juliana");
```

O que acontece por trás:

- Primeiro `a` é `{ nome: "João", media: 10 }` → `a.nome === "Juliana"` → `false`
- Depois `a` é `{ nome: "Juliana", media: 8 }` → `a.nome === "Juliana"` → `true`  
    ✅ Então o método `find()` retorna esse objeto.

#### O que significa `a.nome`?

É apenas a forma de **acessar a propriedade** `nome` de cada objeto `a`.

Se `a = { nome: "Caio", media: 7.5 }`, então:
-  `a.nome` → `"Caio"`
- `a.media` → `7.5`

#### O que significa ` === `?

O ` === ` é o **operador de comparação estrita** do JavaScript.

Ele verifica o **valor e tipo** ao mesmo tempo.

**Exemplo:**

```js
5 == "5"  // true (compara só o valor, conveter tipo automaticamente)
5 === "5" // false (valor igual, mas tipos diferentes: number vs string)
5 === 5   // true (valor e tipo iguais)
```

> Por isso, em JavaScript moderno, usamos sempre ` === ` para evitar erros de conversão automática de tipos (coerção).

#### Colocando tudo junto

No exemplo:

```js
a => a.nome === aluno
```

Significa:
- Para cada elemento `a` do array...
- Pegue a propriedade `nome` desse objeto...
- Compara com a variável `aluno` usando comparação estrita (` === `).

Se for igual, retorne `true`, senão retorne `false`.

## 2. Usando `filter()`

O `filter()` retorna todos os elementos que correspondem a uma condição.

```js
const resultado = alunos.filter(a => a.nome.startsWith("J"));
console.log(resultado);
// [{ nome: "João", media: 10 }, { nome: "Juliana", media 8 }]
```

## 3. Usando `findIndex()`

Semelhante ao `indexOf()`, mas funciona bem com objetos.

```js
const indice = alunos.findIndex(a => a.nome === "Ana");
console.log(indice); // 3
```

# Exemplos práticos de uso

## 1. Procurando um produto em uma lista de compras

```js
const produtos = ["Arros", "Feijão", "Macarrão"];

if(produtos.includes("Feijão")) {
	console.log("Feijão está na lista!")
} else {
	console.log("Feijão não foi encontrado.");
}
```

## 2. Verificando presença de um número

```js
const numeros = [10, 20, 30, 40];
console.log(numeros.includes(30));  // true
console.log(numeros.indexOf(30));   // 2
console.log(numeros.indexOf(50));   // -1 pois 50 não existe na lista
```

## 3. Procurando alunos com média maior que 8

```js
const aprovados = alunos.filter(a => a.media > 8);
console.log(aprovados);
// [ { nome: "João", media: 10 }, { nome: "Ana", media: 9 } ]
```

---
# Conclusão

- **`includes()`** → verifica se o valor existe.
- **`indexOf()`** → retorna a posição do valor.
- **`find()`** → retorna o primeiro objeto que atende à condição.
- **`findIndex()`** → retorna o índice do objeto que atende à condição.
- **`filter()`** → retorna todos os valores que atendem à condição.

✅ Usar o método correto depende da estrutura dos dados (array simples ou objetos) e da necessidade (achar 1 valor ou vários).
