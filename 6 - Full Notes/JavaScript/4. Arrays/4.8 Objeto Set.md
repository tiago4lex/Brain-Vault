# Introdução

## O que é um `Set`

Um `Set` é uma **estrutura de dados nativa** do JavaScript que **armazena valores únicos**, ou seja, **não permite elementos duplicados**.

> É como um array, mas não aceita valores repetidos e não tem índices numéricos.

---
# Criando um `Set`

É possível criar um `Set` usando o construtor `Set()`:

```js
const numeros = new Set();
```

Também pode inicializar com valores:

```js
const numeros = new Set([1, 2, 3, 4]);
console.log(numeros); // Set(4) {1, 2, 3, 4}
```

> Os valores duplicados são automaticamente removidos.

---
# Métodos Principais de `Set`

| Método          | Descrição                                | Exemplo                              |
| --------------- | ---------------------------------------- | ------------------------------------ |
| `add(valor)`    | Adiciona um valor ao Set                 | `meuSet.add(10)`                     |
| `delete(valor)` | Remove um valor do Set                   | `meuSet.delete(10)`                  |
| `has(valor)`    | Verifica se o valor existe               | `meuSet.has(10)` → `true` ou `false` |
| `clear()`       | Remove todos os valores do Set           | `meuSet.clear()`                     |
| `size`          | Retorna a quantidade de elementos no Set | `meuSet.size`                        |

---
# Exemplo Completo

```js
const numeros = new Set();

// Adicionando valores
numeros.add(1);
numeros.add(2);
numeros.add(3);
numeros.add(2); // ignorado (valor duplicado)

console.log(numeros); // Set(3) {1, 2, 3}
console.log(numeros.size); // 3

// Verificando valores
console.log(numeros.has(2)); // true
console.log(numeros.has(5)); // false

// Removendo valores
numeros.delete(2);
console.log(numeros); // Set(2) {1, 3}

// Limpando o set
numeros.clear();
console.log(numeros); // Set(0) {}
```

# Iterando sobre um `Set`

É possível percorrer um `Set` usando:

### `for...of`

```js
const cores = new Set(["azul", "vermelho", "verde"]);

for (const cor of cores) {
	console.log(cor);
}
// azul, vermelho, verde
```

### `forEach`

```js
cores.forEach(cor => console.log(cor))
```

---
# Convertendo `Set` em Array

Como `Set` não usa índices, às vezes é útil transform=alo em um array.

```js
const numeros = new Set([1, 2, 3]);
const arrayNumeros = [...numeros];

console.log(arrayNumeros); // [1, 2, 3]
```

---
# Quando Usar `Set`

Use `Set` quando precisar de uma coleção de valores únicos, por exemplo:
- Remover elementos duplicados de um array.
- Verificar presença de itens com rapidez.
- Armazenar listas de itens sem se preocupar com ordem ou repetição.

---
# Exemplo: Removendo duplicados de um array

```js
const numerosa = [1, 2, 2, 3, 4, 4, 5];

const unicos = [...new Set(numeros)];

console.log(unicos); // [1, 2, 3, 4, 5]
```

---
# Resumo

- `Set` é uma coleção de valores únicos.
- Não aceita elementos duplicados.
- Não possui índices.
- Possui métodos próprios como `add`, `delete`, `has` e `clear`.
- Pode ser iterado e convertido para array facilmente.
