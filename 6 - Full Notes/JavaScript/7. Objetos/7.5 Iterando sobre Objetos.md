# O que significa iterar sobre um objeto

**Iterar** significa **percorrer cada elemento** (ou propriedade) de ume estrutura de dados executando aum ação para cada item.

No caso de objetos em JavaScript, a iteração serve para **percorrer as propriedades (chaves) e seus valores**.

Objetos são compostos por **pares chaves-valor**, como neste exemplo:

```js
const pessoa = {
	nome: 'José',
	idade: 23,
	cidade: 'São Paulo'
};
```

É possível acessar todas as propriedades e seus valores de forma dinâmica (sem saber os nomes antes), para isso é preciso usar técnicas de **iteração sobre objetos**.

---
# Formas de iterar sobre objetos em JavaScript

Existem diversas formas de iterar sobre objetos. As principias são:

1. `for...in`
2. `Object.keys()`
3. `Object.values()`
4. `Object.entries()`
5. Métodos modernos com `forEach()` e `for...of` (em conjunto com os métodos acima)

---
# Estrutura `for...in`

O `for...in` é o método **tradicional** para percorrer **todas as propriedades enumeráveis** de um objeto.

## Sintaxe

```js
for (let chave in objeto) {
	// código executado para cada chave
}
```

## Exemplo Prático

```js
const aluno = {
	nome: 'José',
	curso: 'Engenharia de Software',
	periodo: '4º'
};

for (let chave in aluno) {
	console.log(chave + ": " aluno[chave]);
}
```

**Saída:**

```text
nome: José
curso: Engenharia de Software
periodo: 4º
```

## Explicação

- `chave` recebe o nome da propriedade a cada iteração.
- `aluno[chave]` retorna o **valor** correspondente à chave.

>[!warning] Cuidado:
>O `for...in` **também percorre propriedades herdadas** do protótipo.
>Por isso, use `hasOwnProperty()` se for necessário apenas as propriedades diretas do objeto:
>
>```js
>for (let chave in aluno) {
>	if (aluno.hasOwnPropertu(chave)) {
>		console.log(`${chave}: ${aluno[chave]}`);
>	}
>}
>```

# Método `Object.keys()`

O método `Object.keys(obj)` retorna um **array contendo todas as chaves** (nomes das propriedades) do objeto:

## Sintaxe

```js
Object.keys(objeto)
```

## Exemplo

```js
const carro = {
	marca: 'Toyota',
	modelo: 'Corolla',
	ano: 2022
};

const chaves = Object.keys(carro);
```

**Saída:**

```css
["marca", "modelo", "ano"]
```

Com esse array, podemos iterar usando `forEach()`:

```js
Object.keys(carro).forEach(chave => {
	console.log(`${chave}: ${carro[chave]}`).
});
```

**Saída:**

```text
marca: Toyota
modelo: Corolla
ano: 2022
```

## Quando usar

- Quando for necessário **apenas as chaves** em formato de array.
- Ideal para manipular dados ou aplicar métodos de array (`map`, `filter`, etc.).

---
# Método `Object.values()`

O método `Object.values(obj)` retorna um **array contendo os valores** das propriedades do objeto.

## Sintaxe

```js
Object.values(objeto);
```

## Exemplo

```js
const produto = {
	nome: 'Notebook',
	preco: 3500,
	garantia: "12 meses"
};

const valores = Object.values(produto);
console.log(valores);
```

**Saída:**

```css
["Notebook", 3500, "12 meses"]
```

## Usando um conjunto com `for...of`:

```js
for (let valor of Object.values(produto)) {
	console.log(valor)
}
```

**Saída:**

```text
Notebook
3500
12 meses
```

# Método `Object.entries()`

O método `Object.entries(obj)` retorna um **array de arrays**, onde cada subarray contém `[chave, valor]`.

## Sintaxe

```js
Object.entries(objeto);
```

## Exemplo

```js
const usuario = {
	nome: 'João',
	idade: 23,
	ativo: true
};

const entradas = Object.entries(usuario);
console.log(entradas);
```

**Saída:**

```css
[["nome", "João"], ["idade", 23], ["ativo", true]]
```

É possível iterar com `for...of`:

```js
for (let [chave, valor] of Object.entries(usuario)) {
	console.log(`${chave}: ${valor}`);
}
```

**Saída:**

```text
nome: João
idade: 23
ativo: true
```

## Quando usar

- Quando é preciso tanto das chaves quanto dos valores ao mesmo tempo.
- Excelente para converter objetos em arrays ou mapear dados dinamicamente.

# Outras formas modernas de iteração

## `for...of` com `Object.keys()`

```js
const pessoa = { nome: 'Maria', idade: 22, cidade: 'São Paulo' };

for (let chave of Object.keys(pessoa)) {
	console.log(`${chave}: ${pessoa[chave]}`);
}
```

## `forEach()` com `Object.entries()`

```js
Object.entries(pessoa).forEach(([chave, valor]) => {
  console.log(`${chave}: ${valor}`);
});
```

## Convertendo para Map (para iteração mais poderosa)

```js
const mapa = new Map(Object.entries(pessoa));
for (let [chave, valor] of mapa) {
	console.log(`${chave}: ${valor}`);
}
```

---
# Comparativo entre as abordagens

| Método             | Retorna                   | Tipo de iteração | Itera herdadas? | Melhor uso                      |
| ------------------ | ------------------------- | ---------------- | --------------- | ------------------------------- |
| `for...in`         | Chaves (strings)          | Tradicional      | ✅ Sim           | Objetos simples                 |
| `Object.keys()`    | Array de chaves           | Moderna          | ❌ Não           | Loops modernos e `forEach()`    |
| `Object.values()`  | Array de valores          | Moderna          | ❌ Não           | Processar dados de propriedades |
| `Object.entries()` | Array de `[chave, valor]` | Moderna          | ❌ Não           | Quando precisa de ambos         |

---
>[!warning] Cuidados e Boas Práticas
>✅ Use `for...in` **apenas para objetos**, não para arrays.
>✅ Prefira `Object.keys()`, `Object.values()` ou `Object.entries()` quando quiser usar métodos de array (`forEach`, `map`, `filter`, etc.).
>✅ Combine `for...in` com `hasOwnProperty()` para evitar propriedades herdadas.
>✅ Evite modificar o objeto durante a iteração (pode causar erros).
>✅ Use `for...of` para iterar arrays, strings, maps e sets (iteráveis).

# Exemplo prático completo

```js
const desenvolvedor = {
  nome: "Maria",
  linguagens: ["JavaScript", "Python", "C"],
  experiencia: 2
};

// for...in
console.log("Usando for...in:");
for (let chave in desenvolvedor) {
  console.log(`${chave}: ${desenvolvedor[chave]}`);
}

// Object.keys()
console.log("\nUsando Object.keys():");
Object.keys(desenvolvedor).forEach(chave => {
  console.log(`${chave}: ${desenvolvedor[chave]}`);
});

// Object.entries()
console.log("\nUsando Object.entries():");
for (let [chave, valor] of Object.entries(desenvolvedor)) {
  console.log(`${chave}: ${valor}`);
}
```

**Saída:**

```text
Usando for...in:
nome: Maria
linguagens: JavaScript,Python,C
experiencia: 2

Usando Object.keys():
nome: Maria
linguagens: JavaScript,Python,C
experiencia: 2

Usando Object.entries():
nome: Maria
linguagens: JavaScript,Python,C
experiencia: 2
```

# Conclusão

Iterar sobre objetos é uma das habilidades mais importantes no JavaScript moderno.  
As diferentes abordagens — `for...in`, `Object.keys()`, `Object.values()`, e `Object.entries()` — oferecem **formas flexíveis e poderosas** de manipular dados de forma dinâmica.

| Objetivo                               | Método ideal                                                  |
| -------------------------------------- | ------------------------------------------------------------- |
| Percorrer todas as chaves de um objeto | `for...in` ou `Object.keys()`                                 |
| Obter apenas os valores                | `Object.values()`                                             |
| Trabalhar com pares chave–valor        | `Object.entries()`                                            |
| Fazer operações funcionais com array   | `Object.keys()` / `Object.entries()` + `forEach()` ou `map()` |
