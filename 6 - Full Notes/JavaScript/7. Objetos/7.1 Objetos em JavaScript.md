# O que é um Objeto?

Em **JavaScript**, um objeto é uma estrutura de dados que **armazena informações em pares de chave e valor** (também chamados de **propriedades**).

Cada chave *(key)* é um **string** que identifica o dado, e cada valor *(value)* pode ser qualquer tipo de dado, incluindo outros objetos ou funções.

Em outras palavras, um objeto representa uma **entidade do mundo real**, com suas características (propriedades) e comportamentos (métodos).

### Exemplo Real

Imagine que se quer representar uma pessoa:

```javascript
const pessoa = {
	nome: 'Maria',
	idade: 23,
	profissao: 'Estudante de Engenharia de Software',
	falar: function() {
		console.log('Olá! Meu nome é ' + this.nome);
	}
};

pessoa.falar(); // Saída: Olá! Meu nome é Maria
```

**Explicação:**

- `nome`, `idade`, `profissao` são **propriedades** do objeto `pessoa`.
- `falar` é um **método**, uma função dentro de um objeto.
- `this` faz referência ao **próprio objeto** onde o método está definido.

----
# Conceito e Definição Técnica

>[!quote] Em termos técnicos:
>Um **objeto** em JavaScript é uma **coleção dinâmica de propriedades**.
>Cada propriedade é **definida** por um **par chave-valor**, onde a chave é uma string (ou símbolo) e o valor pode ser qualquer tipo de dado.

Os objetos são a **base da programação orientada a objetos (POO)** em JavaScript. Inclusive, quase tudo em JavaScript é um objeto, como arrays, funções, e até tipos primitivos podem ser tratados como objetos temporariamente (ex.: `"texto".length`).

---
# Sintaxe Básica

A forma mais comum de criar um objeto é usando **literais de objeto** (chave `{}`):

```js
const objeto = {
	chave1: valor1,
	chave2: valor2,
	chave3: valor3
};
```

**Exemplo:**

```js
const carro = {
	marca: 'Toyota',
	modelo: 'Corolla',
	ano: 2022
};
```

**Acessando propriedades**

```js
console.log(carro.marca)      // Toyota
console.log(carro['modelo']); // Corolla
```

**Modificando Propriedades**

```js
carro.ano = 2023     // Atualiza
carro.cor = "Prata"  // Adiciona nova propriedade
delete carro.modelo  //  Remove propriedade
```

# Criando Objetos de Diferente Maneiras

## 1. Usando sintaxe literal

A mais usada e simples:

```js
const usuario = {
	nome: 'Maria',
	idade: 23
};
```

## 2. Usando o construtor `new Object()`

```js
const usuario = new Object();
usuario.nome = 'Maria';
usuario.idade = 23;
```

## 3. Usando uma função construtora

Forma usada antes da introdução de classes no ES6:

```js
function Pessoa(nome, idade) {
	this.nome = nome;
	this.idade = idade;
	this.falar = function() {
		console.log('Olá, eu sou ' + this.nome);
	};
}

const p1 = new Pessoa('Maria', 23);
p1.falar(); //  Olá, eu sou Maria
```

## 4. Usando classes (forma moderna)

As **classes** são uma **forma mais clara e moderna** de criar objetos (desde o ECMAScript 2015):

```js
class Pessoa {
	constructor(nome, idade) {
		this.nome = nome;
		this.idade = idade;
	}
	
	falar() {
		console.log(`Olá, meu nome é ${this.nome}`);
	}
}

const maria = new Pessoa('Maria',23);
maria.falar(); // Olá, meu nome é Maria
```

# Métodos úteis com objetos

O JavaScript possui vários métodos integrados para manipular objetos:

|Método|Descrição|
|---|---|
|`Object.keys(obj)`|Retorna um array com as **chaves** do objeto|
|`Object.values(obj)`|Retorna um array com os **valores**|
|`Object.entries(obj)`|Retorna um array de **pares [chave, valor]**|
|`Object.assign(destino, origem)`|Copia propriedades de um objeto para outro|
|`Object.freeze(obj)`|Impede modificações no objeto|
|`Object.seal(obj)`|Impede adição/remoção, mas permite alterar valores|

**Exemplo:**

```js
const aluno = { nome: "Maria", curso: "Engenharia", idade: 23 };

console.log(Object.keys(aluno));   // ["nome", "curso", "idade"]
console.log(Object.values(aluno)); // ["Maria", "Engenharia", 23]
console.log(Object.entries(aluno));// [["nome", "Maria"], ["curso", "Engenharia"], ["idade", 23]]
```

# Quando usar Objetos?

Você deve usar objetos quando:

- Precisa **representar entidades reais** (ex: usuários, carros, produtos, pedidos...);    
- Precisa **agrupar dados relacionados** em uma estrutura única;    
- Deseja **criar modelos reutilizáveis** (por meio de classes ou funções construtoras);    
- Precisa **manter estado e comportamento** juntos (dados + funções).

Em suma, objetos são ideais quando você quer **organizar o código de forma estruturada e descritiva**.

# Objetos Aninhados *(Nested Objects)*

Objetos podem conter outros objetos dentro deles:

```js
const empresa = {
  nome: "TechCorp",
  endereco: {
    rua: "Rua Central",
    numero: 100,
    cidade: "São Paulo"
  },
  funcionarios: ["Maria", "Ana", "Lucas"]
};

console.log(empresa.endereco.cidade); // São Paulo
console.log(empresa.funcionarios[1]); // Ana
```

# Métodos e `this`

O `this` faz referência ao **objeto atual** em que o método está sendo chamado.

```js
const conta = {
	dono: 'Maria',
	saldo: 1000,
	depositar(valor) {
		this.saldo += valor;
	},
	verSaldo() {
		console.log(`Saldo de ${this.dono}: R$ ${this.saldo}`);
	}
};

conta.depositar(500);
conta.verSaldo(); // Saldo de Maria: R$ 1500
```

---
# Iterando sobre Objetos

É possível percorrer as propriedades de um objeto com o `for...in`:

```js
const produto = {
	nome: 'Notebook',
	preco: 3500,
	marca: 'Dell'
};

for(let chave in produto) {
	console.log(chave + ': ' + produto[chave]);
}
```

**Saída:**

```text
nome: Notebook
preco: 3500
marca: Dell
```

# Comparação com Arrays

|Característica|Objeto|Array|
|---|---|---|
|Estrutura|Chave-Valor|Índice-Valor|
|Ordem dos elementos|Não garantida|Garantida|
|Quando usar|Dados descritivos|Coleções ordenadas|

**Exemplo simples:**

```js
const json = '{"nome": "Maria", "idade": 23}';
const obj = JSON.parse(json); // converte JSON → objeto
console.log(obj.nome); // Maria

const novoJson = JSON.strigify(obj) // converte objto → JSON
console.log(novoJson); // {"nome":"Maria", "idade":23}
```

# Exemplo Prático Completo

```js
// Representando um usuário com métodos
const usuario = {
  nome: "Maria",
  idade: 23,
  email: "maria@email.com",
  logar() {
    console.log(`${this.nome} acabou de logar no sistema.`);
  },
  atualizarEmail(novoEmail) {
    this.email = novoEmail;
    console.log(`Email atualizado para: ${this.email}`);
  }
};

// Testando o objeto
usuario.logar();
usuario.atualizarEmail("novoemail@teste.com");
console.log(usuario);
```

**Saída:**

```text
Maria acabou de logar no sistema.
Email atualizado para: novoemail@teste.com
{ nome: 'Maria', idade: 23, email: 'novoemail@teste.com', ... }
```

# Conclusão

Os **objetos** são uma das estruturas centrais e mais poderosas do JavaScript.

Eles permitem **organizar dados e comportamentos** de forma clara, eficiente e escalável, sendo fundamentais tanto para **programação funcional** quanto para **orientada a objetos**.
