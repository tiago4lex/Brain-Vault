# Variáveis

## O que são?

Variáveis são "caixas" na memória do computador onde podemos armazenar valores para serem usados e manipulados no programa.

Em JavaScript, as varáveis podem ser declaradas usando:

- `var` - escopo de função (evitar no código moderno).
- `let` - escopo de bloco (recomendado para varáveis que mudam).
- `const` - escopo de bloco, valor constante (não pode ser reatribuído).

## Como funciona?

Quando se cria uma varável, o JavaScript reserva um espaço na memória e associa um nome a ele. Esse espaço pode conter diferentes tipos de dados.

## Exemplos

```javascript
// Declarando variáveis
var nome "João";    // método antigo
let idade = 23;     // pode mudar
const PI = 3.14159; // não pode mudar

// Reatribuindo valores
idade = 24;         // permitido
// PI = 3.14; // ERRO: const não pode mudar

console.log(nome, idade, PI);
```

# Tipos de Dados

JavaScript é **dinamicamente tipado**, ou seja, o tipo é atribuído automaticamente com base no valor.

## Tipos Primitivos

1. **String** → Texto (`"Olá"`, `"Mundo"`, `Template`)
2. **Number** → Números inteiros ou decimais (`10`, `3.14`)
3. **Boolean** → Lógico (`true`, `false`)
4. **Undefined** → Variável declarada mas sem valor
5. **Null** → Ausência intencional do valor
6. **Symbol** → Identificadores únicos
7. **Biglnt** → Números inteiros muito grandes

## Tipo Objeto

- Estruturas complexas (arrays, funções, objetos literais).

## Exemplos

```javascript
let texto = "JavaScript";    // String
let numero = 42;             // Number
let ativo = true;            // Boolean
let semValor;                // Undefined
let vazio = null;            // Null
let id = Symbol("id");       // Symbol
let numeroGrande = 123456789012345678901234567890n; // BigInt

let pessoa = { nome: "João", idade: 23 };   // Objeto
let lista = [1, 2, 3, 4];                   // Array

console.log(typeof texto);                  // String
console.log(typeof numeroGrande)            //bigint
```

# Operadores

# O que são?

Operadores são símbolos que realizam operações sobre valores ou variáveis.

## Tipos de Operadores

### Aritméticos

- `+` (adição)
- `-` (subtração)
- `*` (multiplicação)
- `/` (divisão)
- `%` (módulo)
- `**` (exponenciação)

```javascript
let a = 10, b = 3;
console.log(a + b);  // 13
console.log(a % b);  // 1
```

### De Atribuição

- ` =` (atribui valor)
- `+=`, `-=`, `*=`, `/=`, etc.

```javascript
let x = 5;
x += 2;          // x = x + 2
console.log(x);  // 7
```

### Lógicos

- `&&` (E lógico)
- `||` (OU lógico)
- `!` (negação)

```javascript
console.log(true && false); // false
console.log(true || true);  // true
console.log(!true);         // false
```

### Operador Ternário

```javascript
let idade = 18;
let status = idade >= 18 ? "Adulto" : "Menor";
console.log(status);  // Adulto
```

# Funções

## O que são?

Funções são blocos de código que executam uma tarefa e podem ser reutilizados.

## Como funcionam?

- Recebem **parâmetros** (opcional).
- Executam um **bloco de código**.
- Podem retornar um valor com `return`

## Tipos de funções em JavaScript

### Função Declarada

```javascript
function saudacao(nome) {
	return `Olá, ${nome}!`;
}

console.log(saudacao("João"));
```

### Função Anônima

```javascript
const some = function (a, b) {
	return a + b;
};
console.log(soma(5, 3));
```

### Arrow Function

```javascript
const multiplicar = (a, b) => a * b;
console.log(multiplicar(4, 2));
```

### Função com Parâmetro Padrão

```javascript
function mensagem(texto = "Sem mensagem") {
	console.log(texto);
}

mensagem("Olá mundo");
mensagem();  // Sem mensagem
```

### Função sem Retorno

```javascript
function logarUsuario(usuario) {
	console.log(`Usuário logado: ${usuario}`);
}

logarUsuario("João");
```
