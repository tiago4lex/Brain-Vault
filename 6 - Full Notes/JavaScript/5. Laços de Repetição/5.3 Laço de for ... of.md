# Introdução

O `for...of` é um laço de repetição introduzido no **ES6 *(ECMAScript 2015)***.

Ele permite **percorrer de forma simples e legível**  os elementos de estruturas iteráveis, como **arrays, strings, Maps, Sets, NodeLists,** entre outros.

> Diferente do `for` tradicional, o `for...of` não usa índices numéricos: ele trabalha diretamente com os **valores**.

---
#  Sintaxe

```javascript
for (variavel of objetoIteravel) {
	// Código a ser executado para cada elemento
}
```

- `variavel` → representa o **valor atual** em cada iteração.
- `objetoIretavel` → é a colocação (array, string, etc.) que será percorrida.

---
# Exemplo com Arrays

```javascript
const numeros = [10, 20, 30];

for (let numero of numeros) {
	console.log(numero);
}
```

**Saída:**

```text
10
20
30
```

> Aqui, `numero` assume diretamente cada valor do array.

---
# Exemplo com Strings

Como strings também são **iteráveis**, podemos percorrer cada caractere:

```javascript
const palavra = "JavaScript";

for (let letra of palavra) {
	console.log(letra);
}
```

**Saída:**

```text
J
a
v
a
S
c
r
i
p
t
```

---
# Exemplo com `Set` (coleção sem valores duplicados)

```js
const conjunto = new Set([1, 2, 3, 3, 2]);

for (let valor of conjunto) {
	console.log(valor);
}
```

**Saída:**

```text
1
2
3
```

> O `Set` elimina duplicatas automaticamente.

---
# Exemplo com `Map`

O `for...of` percorre pares `[chave, valor]` em um `Map`.

```js
const map = new Ma([
	["nome", "Maria"],
	["idade", "23"],
	["curso", "Direitor"]

for (let [chave, valor] of mapa) {
	console.log(chave, ":", valor);

```

```text

```

# Comparação com outros laços

## `for` tradicional

```js
const numeros = [10, 20, 30];

for (let i = 0; i < numeros.length; i++) {
	console.log(numeros[i]);  // precisa acessar pelo índice
}
```

## `for...in`

```js
const numeros = [10, 20, 30];

for (let i in numeros) {
	console.log(i);  // imprime os índices (0, 1, 2)
}
```

## `for...of`

```js
const numeros [10, 20, 30];

for (let numero of numeros) {
	console.log(numero); // imprime os valores (10, 20, 30)
}
```

## Resumindo

- **for** → ideal para controle por índices.
- **for...in** → percorre **índices** de arrays ou propriedades de objetos.
- **for...of** → percorre **valores** de arrays, strings, Maps, Sets.
