# Introdução

O `forEach()` é um método de arrays em JavaScript que permite **executar uma função para cada elemento de um array.**

Ele é uma forma mais **moderna e legível** de percorrer arrays quando comparado com o `for` tradicional.

>[!note] Nota:
>Diferente do `for` ou do `for...of`, o `forEach()` não cria um laço de repetição manual:
>- Ele **chama automaticamente** uma função *callback* para cada elemento do array.
>- Essa função pode receber até **três parâmetros:**
>	- **valor** (elemento atual)
>	- **índice** (posição do elemento)
>	- **array** (o próprio array que está sendo percorrido)

---
# Sintaxe

```js
array.forEach(function(elemento, indice, array){
	// código a ser executado para cada elemento
});
```

- `elemento`: valor atual do array na iteração.
- `índice`: posição do elemento no array.
- `array`: o array inteiro que está sendo percorrido.

---
# Exemplo básico

```js
const numeros = [1, 2, 3];

numeros.forEach(function(numero){
	console.log(numero);
});
```

**Saída:**

```text
1
2
3
```

---
# Exemplo com Arrow Function

Como o `forEach()` recebe uma função, podemos usar uma ***arrow function*** para deixar o código mais limpo:

```js
const numeros = [10, 20, 30];

numero.forEach(numero => console.log(numero));
```

**Saída:**

```text
10
20
30
```

---
# Usando o índice e o Array

```js
const frutas = ["Maçã", "Banana", "Uva"];

frutas.forEach((fruta, indice, array) => {
	console.log(`${fruta} está na posição ${indice} do array [${array}]`);
});
```

**Saída:**

```php
Maçã está na posição 0 do array [Maçã,Banana,Uva]
Banana está na posição 1 do array [Maçã,Banana,Uva]
Uva está na posição 2 do array [Maçã,Banana,Uva]
```

---
# Calculando a soma com `forEach()`

```js
const numeros = [5, 10, 15];
let soma = 0;

numeros.forEach(numero => {
	soma += numero;
});

console.log("Soma:", soma);
```

**Saída:**

```text
Soma: 30
```

>[!note] Nota:
>Aqui usamos `soma += numero` dentro do `forEach()` para acumular os valores.

# Diferença entre `forEach()` e outros laços

## `for`

```js
const numeros = [1, 2, 3];
for (let i = 0; i < numeros.length; i++) {
	console.log(numeros[i]);
}
```

## `for..of`

```js
const numeros = [1, 2, 3];
for (let numero of numeros) {
	console.log(numero);
}
```

## `forEach()`

```js
const numeros = [1, 2, 3];
numeros.forEach(numero => console.log(numero));
```

## Resumo

- `for` → mais flexível, mas mais verboso.
- `for...of` → mais simples quando só queremos os valores.
- `forEach()` → ótimo para trabalhar com arrays de forma funcional.

---
# Limitações do `forEach()`

- **Não pode ser interrompido**:  
    Diferente do `for` e do `for...of`, não é possível usar `break` ou `continue` dentro de um `forEach()`.  
    Se precisar interromper, use `for` ou `for...of`.
    
- **Não retorna valor**:  
    O `forEach()` sempre retorna `undefined`.  
    Se você quiser gerar um novo array a partir de outro, use métodos como `map()`, `filter()` ou `reduce()`.

---
# Conclusão

- O `forEach()` é um método de arrays que **executa uma função callback para cada elemento**.
- Ele é ideal para casos onde **queremos apenas percorrer e executar ações**, sem a necessidade de retornar um novo array.
- Muito usado junto com **arrow functions** para deixar o código mais limpo.
- Não pode ser interrompido (`break`/`continue`) e não retorna valores.
