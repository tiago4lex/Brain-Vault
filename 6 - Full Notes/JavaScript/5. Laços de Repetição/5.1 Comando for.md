# Introdução

## O que é o `for`?

O `for` é um dos laços de repetição mais usados em JavaScript.

Ele é ideal quando sabemos **exatamente quantas vezes** queremos que um bloco de código seja executado.

Em vez de repetir manualmente várias linhas, o `for` automatiza essa repetição de forma **controlada e eficiente**.

---
# Estrutura básica

O `for` possui três partes principais:

```js
for (inicialização; condição; incremento) {
	// Bloco de código a ser executado
}
```

## Explicação dos parâmetros

1. **Inicialização:** geralmente usada para criar e inicializar uma variável de controle (ex: `let i = 0`).
2. **Condição:** define até quando o loop será executado (enquanto for verdadeira).
3. **Incremento/Decremento:** atualização da variável de controle a cada iteração (ex: `i++`).

---
# Exemplos simples

```js
for (let i = 0; i < 5; i++) {
	console.log("Número:", i);
}
```

**Saída:**

```makefile
Número: 0
Número: 1
Número: 2
Número: 3
Número: 4
```

---
# Variações do `for`

## Decremento

```js
for (let i = 5; i > 0; i--) {
	console.log(i);
}
// Saída: 5, 4, 3, 2, 1
```

## Passos diferentes (incremento personalizado)

```js
for (let i = 0; i <=10; i +=2) {
	console.log(i);
}
// Saída: 0, 2, 4, 6, 8, 10
```

## Inicialização e incremento fora do `for`

```js
let i = 0;
for (; i < 3;) {
	console.log(i);
	i++;
}
// Saída: 0, 1, 2
```

---
# O `for` em arrays

O `for` é muito usado para **percorrer listas**:

```js
const frutas = ["Maçã", "Banana", "Laranja"];

for (let i = 0; i < frutas.length; i++) {
	console.log(`Frutas ${i}: ${frutas[i]}`);
}
```

**Saída:**

```
Fruta 0: Maçã
Fruta 1: Banana
Fruta 2: Laranja
```

---
# Controle com `break` e `continue`

## `break` → encerra o laço

```js
for (let i = 1; i <=5; i++) {
	if (i === 3) break;
	console.log(i);
}
// Saída: 1, 2
```

## `continue` → pula para a próxima iteração

```js
for (let i = 1; i<=5; i++) {
	if (i === 3) continue;
	console.log(i);
}
// Saída: 1, 2, 3, 4, 5
```

---
# Uso prático: somar números de uma lista

```js
const numeros = [10, 20, 30, 40];
let soma = 0;

for (let i = 0; i < numeros.length; i++) {
	soma += numeros[i];
}

console.log("Soma total:", soma);
// Saída: Soma total: 100
```

# Resumo

- ✅ Ideal quando o número de repetições é conhecido.
    
- ✅ Estrutura: **inicialização → condição → incremento**.
    
- ✅ Pode ser usado para percorrer arrays, fazer cálculos, validar condições.
    
- ✅ Pode usar `break` e `continue` para controlar o fluxo.
