# Introdução

Dicionários são estruturas de dados usadas para armazenar informações no formato **chave : valor**.

Eles são uma das estruturas **mais importantes e mais usadas** em Python, pois permitem representar dados do mundo real de forma clara e eficiente.

Principais características:

- Estruturas chave → Valor
- Chaves únicas e imutáveis
- Valores podem ser de qualquer tipo
- Mutáveis (podem ser alterados)
- Muito rápidos para busca

---
# Conceito de Chave e Valor

- **Chave *(Key)*:** identificador único
- **Valor *(value)*:** dado associado à chave

## Exemplos simples

```python
pessoa = {
	"nome": "Maria",
	"idade": 23,
	"curso": "Engenharia de Software"
}
```

Aqui:

- `"nome"`, `"idade"` e `"curso"` são as chaves.
- Os dados associados são os valore.

---
# Criação de Dicionários

## 1. Forma literal

```python
dicionario = {"a": 1, "b": 2}
```

## 2. Usando `dict()`

```python
dados = dict(nome="Maria", idade=23)
```

## 3. Dicionário vazio

```python
d = {}
```

---
# Acesso e Modificação de Valores

## 1. Acesso direto

```python
print(pessoa["nome"])
```

>[!warning] Atenção:
>Se a chave não existir, ocorre erro `KeyError`.


## 2. Usando `get()` (mais seguro)

```python
print(pessoa.get("email", "Não informado"))
```

## 3. Adicionando ou alterando valores

```python
pessoa["email"] = "maria@email.com"
pessoa["idade"] = 23
```

---
# Métodos Principais de Dicionários

| **Método** | **Descrição**             |
| ---------- | ------------------------- |
| `keys()`   | Retorna todas as chaves   |
| `values()` | Retorna todos os valores  |
| `items()`  | Retorna pares chave-valor |
| `get()`    | Acessa valor sem erro     |
| `update()` | Atualiza dados            |
| `pop()`    | Remove chave              |
| `clear()`  | Limpa o dicionário        |

## Exemplo com `update()`

```python
pessoa.update({"cidade": "São Paulo"})
```

# Iteração em Dicionários

## 1. Iterando pelas chaves

```python
pessoa = {
	"nome": "Maria",
	"cidade": "São Paulo",
	"curso": "Engenharia de Software"
}

for chave in pessoa:
	print(chave)
```

**Resultado:**

```text
nome
cidade
curso
```

## 2. Iterando pelos valores

```python
for valor in pessoa.values():
	print(valor)
```

**Resultado:**

```text
Maria
São Paulo
Engenharia de Software
```

## 3. Iterando por chave e valor

```python
for chave, valor in pessoa.items():
	print(chave, "->", valor)
```

**Resultado:**

```text
nome -> Maria
cidade -> São Paulo
curso -> Engenharia de Software
```

# Diferença entre `dict`, `list` e `set`

| **Estrutura** | **Uso Principal**          |
| ------------- | -------------------------- |
| `list`        | Coleções ordenadas         |
| `set`         | Valores únicos             |
| `dict`        | Relacionamento chave-valor |

---
# Erros Comuns

❌ Acessar chave inexistente sem `get()`
❌ Usar listas como chaves
❌ Confundir `keys()` com valores

---
# Boas Práticas

- Use chaves claras e descritivas
- Prefira `get()` para acesso seguro
- Use dicionários aninhados com moderação
