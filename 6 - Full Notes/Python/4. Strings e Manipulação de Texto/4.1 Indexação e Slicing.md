# Introdução

Em Python, **indexação e slicing** são conceitos fundamenteis para acessar partes específicas de **sequências**, como:

- Strings (`str`)
- Listas (`list`)
- Tuplas (`tuple`)
- Outros objetos iteráveis

Dominar esses conceitos permite manipular dados com precisão, criar lógicas eficientes e escrever código mais limpo e profissional.

---
# O que é Indexação?

Indexação é a forma de **acessar um único elemento** de uma sequência usando sua posição (índice).

>[!warning] Importante:
>- Em Python, a indexação começa em 0.

---
# Indexação Positiva

## Exemplo com String

```python
texto = "Python"

print(texto[0]) # P
print(texto[1]) # y
print(texto[2]) # t
print(texto[3]) # h
print(texto[4]) # o
print(texto[5]) # n
```

**Explicação:**

- `texto[0]` acessa o primeiro caractere.
- `texto[5]` acessa o último caractere.

## Exemplo com Lista

```python
numeros = [10, 20, 30, 40]

print(numeros[0]) #10
print(numeros[2]) #30
```

---
# Indexação Negativa

A indexação negativa permite acessar elementos **a partir do final da sequência**.

- `-1` → último elemento
- `-2` → penúltimo elemento

## Exemplo

```python
texto = "Python"

print(texto[-1]) # n
print(texto[-2]) # o
```

**Uso comum:**

- Acessar rapidamente o último elemento de uma sequência.

---
# O que é Slicing?

Slicing permite acessar **uma parte (subsequência)** de uma sequência.
Em vez de um único elemento, é possível obter um **intervalo de elementos**.

---
# Sintaxe do Slicing

```python
sequencia[inico:fim:passo]
```

- `inicio` → índice inicial (inclusivo)
- `fim` → índice final (exclusivo)
- `passo` → intervalo entre os elementos

>[!warning] Atenção:
>- Todos os parâmetros são opcionais.

---
# Slicing Básico

## Exemplo com String

```python
texto = "Python"

print(texto[0:4]) # Pyth
print(texto[2:6]) # thon
```

**Explicação:**

- O índice final **não é incluído**.

## Exemplo com Lista

```python
numeros = [0, 1, 2, 3, 4, 5]

print(numeros[1:4]) # [1, 2, 3]
```

---
# Omitindo Valores no Slicing

## Omitir o Início

```python
texto = "Python"
print(text[:3]) # Pyt
```

## Omitir o Fim

```python
print(texto[3:]) # hon
```

## Omitir Ambos

```python
print(texto[:]) # Python
```

---
# Slicing com Passo

O parâmetro `passo` define de **quantos em quantos índices** os elementos serão selecionados.

## Exemplo

```python
numeros = [0, 1, 2, 3, 4, 5]

print(numeros[::2]) # [0, 2, 4]
```

---
# Slicing Reverso

É possível inverter uma sequência usando passo negativo.

## Exemplo

```python
texto = "Python"

print(texto[::-1]) # nohtyP
```

**Uso comum:**

- Inverter strings.
- Verificar palíndromos.

---
# Indexação e Slicing e Tuplas

Tuplas funcionam da mesma forma que listas, porém são **imutáveis**.

```python
tupla = (10, 20, 30, 40)

print(tupla[1])   # 20
print(tupla[1:3]) # (20, 30)
```

---
# Erros Comuns

❌ Acessar índice inexistente

```python
texto = "Python"

print(texto[10]) # IndexError
```

❌ Confundir índice com posição humana

---
# Boas Práticas

- Sempre valide o tamanho da sequência
- Use slicing para evitar loops desnecessários
- Prefira slicing para cópias de listas

Exemplo:

```python
copia = numeros[:]
```

---
# Comparação Rápida

| **Conceito** | **Retorno**       |
| ------------ | ----------------- |
| Indexação    | Um único elemento |
| Slicing      | Uma subsequência  |
