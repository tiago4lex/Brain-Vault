# Introdução

Strings (`str`) representam **texto** em Python e são um dos tipos de dados mais utilizados. Python oferece diversos **métodos nativos** para manipulação de strings, permitindo limpar dados, validar entradas, formatar textos e muito mais.

>[!warning] Importante:
>- Strings são **imutáveis** → nenhum método altera a string original
>- Os métodos sempre retornam **uma nova string**

---
# Métodos de Transformação de Texto

## `upper()` e `lower()`

Convertem o texto para letras maiúsculas ou minúsculas.

```python
texto = "Python"

print(texto.upper()) # PYTHON
print(texto.lower()) # python
```

Uso comum:

- Padronização de dados
- Comparações sem diferença de maiúsculas/minúsculas

## `capitalize()`

Coloca a **primeira letra em maiúscula**.

```python
texto = "python é incrível"
print(texto.capitalize())
```

## `title()`

Coloca a primeira letra de **cada palavra** em maiúsculo.

```python
texto = "python é incrível"
print(texto.title())
```

---
# Métodos de Remoção de Espaços

## `strip()`

Remove espaços do **início e do fim** da string.

```python
texto = " Python "
print(texto.strip())
```

## `lstrip` e `rstrip()`

- `lstrip()` → remove espaços da esquerda
- `rstrip()` → remove espaços da direita

```python
texto = " Python "

print(texto.lstrip())
print(texto.rstrip())
```

---
# Métodos de Busca e Verificação

## `find()`

Retorna o índice da primeira ocorrência do texto.

```python
texto = "Python é incrível"
print(texto.find("é"))
```

> Retorna `-1` se não encontrar.

## `count()`

Conta quantas vezes um valor aparece.

```python
texto = "banana"
print(texto.count("a")) # 3
```

## `startswith()` e `endswith()`

Verificam como a string começa ou termina.

```python
texto = "arquivo.txt"
print(texto.startswith("arquivo"))
print(texto.endswith(".txt"))
```

---
# Métodos de Substituição

## `replace()`

Substitui partes da string.

```python
texto = "Python é difícil"

print(texto.replace("difícil", "incrível"))
```

---
# Métodos de Divisão e Junção

## `split()`

Divide a string em uma lista.

```python
texto = "Python é muito bom"

print(texto.split())
# ['Python', '�', 'muito', 'bom']
```

## `join`

Une elementos de uma lista em um string.

```python
palavras = ["Python", "é", "top"]
print(" ".join(palavras))
# Python é top
```

---
# Métodos de Validação

## `isdigit()`

Verifica se a string contém apenas números.

```python
print("123".isdigit()) # True
```

## `isalpha()`

Verifica se contém apenas letras.

```python
print("Python".isalpha()) # True
```

## `isalnum()`

Verifica se contém letras e números

```python
print("Python3".isalnum()) # True
```

## `isspace()`

Verifica se contém apenas epaços.

```python
print("   ".isspace())
```

# Métodos de Formatação

## Métodos de Formatação

## `format()`

```python
nome = "João"

print("Olá, {}".format(nome))
```

## f-strings (forma moderna)

```python
nome = "João"
idade = 23
print(f"{nome} tem {idade} anos")
```

# Comparação de Strings

```python
senha = "Admin"

if senha.lower() == "admin":
	print("Senha correta")
```

# Erros Comuns

❌ Tentar modificar uma string diretamente

```python
texto[0] = "P" # Erro
```

❌ Esquecer que métodos retornam nova string

---
# Boas Práticas

- Sempre normalize strings antes de comparar
- Use métodos de validação para entrada do usuário
- Prefira f-strings para formatação
