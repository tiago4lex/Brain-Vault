# Introdução

Em Python, **strings multilinha** permitem trabalhar com textos que ocupam **mais de uma linha**, preservando quebras de linha, espaços e formatação. Esse recurso é muito útil para:

- Textos longos
- Mensagens formatadas
- Templates
- Documentação
- Logs e banners
- SQL, HTML, ou JSON embutidos no código.

---
# O que são Strings Multilinha?

Strings multilinha são strings definidas usando **três aspas:**

- Três aspas simples:`'''`
- Três aspas duplas: `"""`

Tudo o que estiver entre essas aspas será considerado parte da string, incluindo **quebras de linha**.

---
# Sintaxe Básica

```python
texto = """
Esta é uma string
que ocupa várias
linhas diferentes.
"""

print(texto)
```

**Resultado:**

```text
Esta é um string
que ocupa várias
linhas diferentes.
```

---
# Aspas Simples vs Aspas Duplas

Ambas funcionam da mesma forma:

```python
texto1 = '''Texto multilinha
com aspas simples'''

texto2 = """Texto multilinha
com aspas duplas"""
```

A escolha é apenas uma **questão de estilo** ou conveniência.

---
# Quebras de Linha (`\n`) vs Strings Multilinha

É possível criar múltiplas linhas usando `\n`, mas strings multilinhas são mais legíveis.

## Usando `\n`

```python
texto = "Linha 1\nLinha 2\nLinha 3"
```

## Usando String Multilinha

```python
texto = """
Linha 1
Linha 2
Linha 3
"""
```

---
# Indentação e String Multilinha

>[!warning] Atenção:
>A indentação dentro do código é preservada na string.

```python
texto = """
	Texto com identação
	será exibido com espaços
"""
```

Isso pode gerar espaços indesejados.

---
# Removendo Indentação Indesejada (`textwrap.dedent`)

Para remover a indentação extra, utilize o módulo `textwrap`.

```python
import textwrap

texto = textwrap.dedent("""
Linha 1
Linha 2
Linha 3
""")

print(texto)
```

---
# Strings Multilinha como Docstrings

Strings multilinha são usadas para criar **docstrings**, que documentam funções, classes e módulos.

## Exemplo

```python
def soma(a, b):
"""
Soma dois números.

Parâmetros:
a (int): primeiro número
b (int): segundo número

Retorna:
int: soma de a e b
"""
return a + b
```

Docstrings podem ser acessadas com:

```python
print(soma.__doc__)
```

# Strings Multilinha em Templates

## Exemplo de mensagens formatada

```python
nome = "Maria"
idade = 23

mensagem = f"""
Olá, {nome}!

Sua idade é {idade} anos.
Seja bem-vindo ao sistema.
"""

print(mensagem)
```

# Uso com SQL, HTML e JSON

## Exemplo com SQL

```python
query = """
SELECT *

"""
```

## Exemplo com HTML

```python
html = """
<html>
	<body>
		<h1>Título</h1>

```

---
# Métodos de String funcionam normalmente

Strings multilinhas são strings comuns, então todos os métodos funcionam:

```python
texto = """
Python
é
incrível
"""

print(texto.upper())
print(texto.split())
```

---
# Erros Comuns

❌ Espaços indesejados por causa da indentação
❌ Esquecer que a quebra de linha inicial conta
❌ Misturar aspas incorretamente

---
# Boas Práticas

- Use strings multilinha para textos longos
- Evite indentação desnecessária
- Use `dedent` quando necessário
- Use docstrings para documentação

---
# Conclusão

Strings multilinha tornam o código Python mais legível e organizado ao lidar com textos extensos. Elas são amplamente usadas em documentação, templates, automação e desenvolvimento profissional.

Dominar esse recurso ajuda a escrever código mais limpo, claro e bem documentado.
