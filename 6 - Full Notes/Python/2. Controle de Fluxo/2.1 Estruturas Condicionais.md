# O que são Estruturas Condicionais?

Estruturas condicionais permite que um programa **tome decisões**  durante a execução. Em vez de executar sempre as mesmas instruções em sequência, o programa pode **escolher caminhos diferentes** dependendo de uma condição ser verdadeira ou falsa.

Em Python, essas decisões são baseadas em **expressões booleans**, ou seja, expressões que resultam em `True` (verdadeiro) ou `False` (falso).

Exemplo do dia a dia:

- Se estiver chovendo → levo guarda chuva.
- Senão → não levo guarda-chuva.

Em python, fazemos isso usando principalmente:

- `if`
- `elif`
- `else`

---
# Operadores usados em condições

Antes de entrar nas estruturas condicionais, é importante entender os operadores mais usados nelas.

## Operadores Relacionais

São usadas para comparar valores:

- ` == ` → igual a
- `!=` → diferente de
- `>` → maior que
- `<` → menor que
- `>=` → maior ou igual
- `<=` → menor ou igual

**Exemplo:**

```python
idade = 18
print(idade >= 18) # True
```

## Operadores Lógicos

Permitem combinar condições:

- `and` → verdadeiro se TODAS as condições forem verdadeiras.
- `or` → verdadeiro se PELO MENOS UMA condição for verdadeira.
- `not` → inverte o valor lógico.

**Exemplo:**

```python
idade = 20
tem_carteira = True

print(idade >= 18 and tem_carteira) # True
```

---
# Estrutura Condicional `if`

A estrutura `if` é usada para executar um bloco de código **apenas se uma condição for verdadeira**.

## Sintaxe Básica

```python
if condição:
	# bloco de código executado se a condição for True
```

>[!warning] Atenção:
>- O bloco é definido pela **indentação**.
>- Não existem chave `{}` como em outras linguagens.

## Exemplo Simples

```python
idade = 20

if idade >= 18:
	print("Você é maior de idade")
```

**Explicação:**

- A condição `idaed >= 18` é válida.
- Como o valor é `True`, o `print` é executado.

---
# Estrutura Condicional `if` + `else`

O `else` é executado **quando a condição do if é falsa**.

## Sintaxe

```python
if condição:
	# Executa se for True
else:
	# Executa se for False
```

## Exemplo Prático

```python
nota = 6

if nota >= 7:
	print("Aprovado")
else:
	print("Reprovado")
```

**Explicação:**

- Se a nota for maior que igual a 7 → aprovado.
- Caso contrário → reprovado.

---
# Estrutura Condicional `if` + `elif` + `else`

Quando temos **mais de duas possibilidades**, usamos `elif` (else if).

## Sintaxe

```python
if condição1:
	# executa se condição1 for True
elif condição2:
	# executa se condição2 for True
else:
	# executa se nenhuma condição for True
```

## Exemplo Detalhado

```python
nota = 8

if nota >=9:
	print("Excelente")
elif nota >=7:
	print("Aprovado")
elif nota >=5:
	print("Recuperação")
else:
	print("Reprovado")
```

Explicação passo a passo:

1. O Python testa `nota >= 9`
2. Se for falso, testa `nota >= 7`
3. Se ainda for falso, testa `nota >= 5`
4. Se nenhuma for verdadeira, executa o `else`

>[!warning] Importante:
>Apenas um bloco será executado.

# Condições Aninhadas

É possível colocar um `if` dentro de outro `if`. Isso é chamado de **condição aninhada**.

## Exemplo

```python
idade = 20
tem_titulo = True

if idade >= 18:
	if tem_titulo:
		print("Pode votar")
	else:
		print("Não pode votar (sem título)")
else:
	print("Menor de idade")
```

**Explicação:**

- Primeiro verifica a idade.
- Depois verifica se a pessoa tem título.

---
# Operador Ternário (Condicional em uma Linha)

Python permite escrever uma condição simples em **uma única linha.**

## Sintaxe

```python
valor_se_verdadeiro if condição else valor_se_falso
```

## Exemplo

```python
idade = 16
status = "Maior de idade" if idade >= 18 else "Menor de idade"

print(status)
```

---
# Boas Práticas com Estruturas Condicionais

- Use nomes de variáveis claros
- Evite condições muito complexas
- Prefira legibilidade a código curto
- Use parênteses para clareza quando necessário
- Evite muitos níveis de aninhamento

---
# Erros Comuns de Iniciantes

❌ Usar ` = ` em vez de ` == `

```python
if x = 10: # errado
```

✅ Correto:

```python
if x == 10:
```

❌ Esquecer os dois pontos `:`
❌ Errar a indentação

## Exemplo 1

Bruno gerencia um pequeno comércio e quer saber qual produto teve o melhor desempenho de vendas no mês passado. Ele registrou a quantidade vendida de dois produtos: maçãs e bananas. Agora, ele precisa escrever um programa que identifique e exiba qual deles teve maior venda.

**Solução:**

```python
macas = int(input("Digite a quantidade de maçãs vendidas: "))
bananas = int(input("Digite a quantiade de maças vendidas: "))

if macas > bananas:
	print("As maças tiveram mais vendas.")
elif bananas < macas:
	print("As bananas tiveram mais vendas.")
else:
	print("As vendas foram iguais.")
```

## Exemplo 2

Camila está organizando um projeto e precisa calcular o tempo total necessário para concluir três atividades: A, B e C. No entanto, se alguma atividade tiver um número de dias negativo, o código deve avisar que os valores inseridos são inválidos e não calcular o total.

```python
atividade_A = int(input("Informe os dias para a atividade A: "))
atividade_B = int(input("Informe os dias para a atividade B: "))
atividade_C = int(input("Informe os dias para a atividade C: "))

if (atividade_A > 0 and atividade_B >= and aitividade_C >= 0):
	tempo_total = atividade_A + atividade_B + atividade_C
	print(f"O tempo total do projeto é de {tempo_total} dias.")
else:
	print("Erro: Os dias não podem ser negativos.")
```

## Exemplo 4

Anna Júlia está criando um sistema para calcular o Índice de Massa Corporal (IMC) e fornecer recomendações básicas. O programa deve receber o peso e a altura de uma pessoa e exibir o valor do IMC, além de indicar se está abaixo do peso, com peso normal ou acima do peso. Crie um programa que receba o peso (em kg) e a altura (em metros) e calcule o IMC usando a fórmula: IMC = peso / (altura ** 2) Depois, exiba o valor do IMC e uma mensagem indicando se está abaixo do peso (IMC < 18.5), peso normal (18.5 <= IMC < 25) ou acima do peso (IMC >= 25).

**Solução:**

```python
peso = float(input("Digite seu peso (kg): "))
altura = float(input("Digite sua altura (m): "))

imc = peso / (altura ** 2)
print(f"Seu IMC é: {imc:.2f}")

if imc < 18.5:
    print("Você está abaixo do peso.")
elif imc < 25:
    print("Você está com peso normal.")
else:
    print("Você está acima do peso.")
```

# Conclusão

As estruturas condicionais são fundamentais em Python e em qualquer linguagem de programação. Elas permitem que o programa seja **dinâmico**, **inteligente** e **responsivo a diferentes situações**.

Dominar bem `if`, `elif` e `else` é essencial para avançar para laços de repetição, funções e projetos mais complexos.
