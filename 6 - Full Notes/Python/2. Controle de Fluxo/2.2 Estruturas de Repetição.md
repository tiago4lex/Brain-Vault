# O que são Estruturas de Repetição?

Estruturas de repetição (ou laços de repetição) permitem que um programa **execute o mesmo bloco de código várias vezes**, enquanto uma condição for satisfeita ou enquanto existirem elementos para percorrer.

Elas são fundamentais para:

- Automatizar tarefas repetitivas.
- Percorrer listas, strings e outros dados.
- Criar contadores, validadores e processamentos em massa.

Em Python, as principais estruturas de repetição são:

- `while`
- `for`

Além disso, existem comandos auxiliares que controlam o comportamento dos laços:

- `range()`
- `break`
- `continue`
- `pass`

---
# Estruturas de Repetição `while`

O laço `while` executa um bloco de código **enquanto uma condição for verdadeira**.

## Sintaxe do `while`

```python
while condição:
	# bloco de código repetido
```

>[!warning] Atenção:
>- Se a condição nunca se tornar falsa, o loop será **infinito**.
>- É essencial garantir que algo dentro do laço altere a condição.

## Exemplo Básico com `while`

```python
contado = 1

while contador <= 5:
	print(contador)
	contador +- 1
```

**Explicação:**

1. A variável `contador` começa com valor 1.
2. A condição `contador <= 5` é verificada.
3. Enquanto for verdadeira, o código é executado.
4. O contador é incrementado até a condição se tornar falsa.

## 3. Exemplo de Validação com `while`

```python
senha = ""

while senha != "1234":
	senha = inpute("Digite a senha: ")
	
print("Acesso permitido")
```

**Uso comum:**

- Validação de entrada do usuário.
- Menus interativos

# Estrutura de Repetição `for`

O laço `for` é usado para **percorrer sequências de elementos**, como listas, strings, tuplas ou intervalos numéricos.

## 1. Sintaxe do `for`

```python
for variavel in sequencia:
	# bloco de código
```

## 2. Exemplo Simples com `for`

```python
for letra in "Python":
	print(letra)
```

**Explicação:**

- O Python percorre cada caractere da string.
- A variável `letra` assume um valor diferente a cada repetição.

## Exemplo com Lista

```python
numeros = [10, 20, 30, 40]

for numero in numeros:
	print(numero)
```

---
# Função `range()`

A função `range()` é muito usada junto com o `for` para gerar **sequências numéricas**.

## Sintaxe do `range()`

```python
range(inicio, fim, passo)
```

- `inicio` → valor inicial (opcional)
- `fim` → valor final (não incluído)
- `passo` → incremento (opcional)

## 2. Exemplos de `range()`

**Exemplo 1:**

```python
for i in range(5):
	print(i)
```

```text
0
1
2
3
4
5
```

**Exemplo 2:**

```python
for i in range(1, 6):
	print(i)
```

```text
1
2
3
4
5
```

**Exemplo 3:**

```python
for i in range(0, 10, 2):
	print(i)
```

```text
2
4
6
8
```

# Comando `break`

O comando `break` **interrompe** completamente o laço, mesmo que a condição ainda seja verdadeira.

## Exemplo com `break`

```python
for numero in range(10):
	if numero == 5:
		break
	print(numero)
```

**Explicação:**

- Quando `numero` chega a 5, o laço é encerrado.

**Saída:**

```text
0
1
2
3
4
```

---
# Comando `continue`

O comando `continue` **interrompe apenas a iteração atual**, pulando para a próxima repetição.

## Exemplo com `continue`

```python
for numero in range(5):
	if numero == 2:
		continue
	print(numero)
```

**Saída:**

```text
1
3
4
```

---
# Comando `pass`

O comando `pass` é um **comando vazio**. Ele não faz nada, mas evita erros de sintaxe quando o Python exige um bloco de código.

## Exemplo com `pass`

```python
for i in range(5):
	for i == 3:
		pass
	print(i)
```

Uso comum do `pass`:

- Estruturas ainda não implementadas.
- Classes e funções vazias.
- Código em desenvolvimento.

# Exemplos práticos

## Exemplo 1

Ana está desenvolvendo um programa que precisa processar uma lista de 5 nomes de clientes para gerar relatórios mensais. Para isso, ela precisa escrever um programa que percorra a lista de nomes e exiba cada cliente.

**Solução:**

```python
clientes = ["João", "Maria", "Carlos", "Ana", "Beatriz"]

for cliente in clientes:
    print(cliente)
```

## Exemplo 2

André está testando um novo recurso no backend do Buscante que processa dados em um loop. Durante os testes, ele percebeu que o sistema parou de responder, e suspeita que o problema está em um loop infinito.

**Solução:**

```python
contador = 0

while contador < 10:
    print("Processando dados...")
    contador += 1  # Atualiza o contador para evitar o loop infinito
```

## Exemplo 3

Marcos está desenvolvendo um programa para exibir uma mensagem de boas-vindas repetidamente no console, como parte de uma campanha de marketing de sua plataforma chamada Buscante. Ele quer garantir que a mensagem seja exibida 5 vezes.

```python
for i in range(5):
    print("Bem-vindo ao Buscante!")
```

## Exemplo 4

Você está recebendo uma lista de valores representando os produtos de sua loja virtual e gostaria de calcular a soma total desses produtos para entender o desempenho financeiro semanal.

**Solução:**

```python
numeros = [10, 20, 30, 40, 50]

soma = 0
for numero in numeros:
    soma += numero

print(f"A soma total das receitas é: {soma}")
```

## Exemplo 5

Ana está desenvolvendo seu portfólio para exibir os projetos de Python que concluiu. Ela organizou uma lista com o nome de cada projeto, mas percebeu que alguns itens podem estar ausentes, aparecendo como `None`.

Crie um programa que ajude Ana a percorrer a lista de projetos e exiba os nomes dos projetos válidos. Se encontrar um item `None`, o programa deve exibir a mensagem: `"Projeto ausente"`.

```python
projetos = ["website", "jogo", "análise de dados", None, "aplicativo móvel"]

for projeto in projetos:
    if projeto is None:
        print("Projeto ausente")
    else:
        print(projeto)
```

## Exemplo 6

José está desenvolvendo uma funcionalidade no sistema do Buscante para interromper a busca assim que um livro específico é encontrado. A lista de livros já cadastrados no sistema é a seguinte:

```python
livros = ["1984", "Dom Casmurro", "O Pequeno Príncipe", "O Hobbit", "Orgulho e Preconceito"]
```

Ajude José a criar um programa que percorra a lista e exiba a mensagem `"Livro encontrado: <nome do livro>"` assim que o livro `"O Hobbit"` for encontrado. Após encontrar o livro, o programa deve parar imediatamente a busca, sem verificar os livros restantes.

**Solução:**

```python
livros = ["1984", "Dom Casmurro", "O Pequeno Príncipe", "O Hobbit", "Orgulho e Preconceito"]

for livro in livros:
    if livro == "O Hobbit":
        print(f"Livro encontrado: {livro}")
        break
```

## Exemplo 7

Você está desenvolvendo um sistema de controle de estoque para o Buscante. Um dos requisitos é verificar a quantidade de exemplares de um livro em estoque e continuar vendendo até que o estoque se esgote. Sempre que uma venda é realizada, o sistema deve informar o usuário e atualizar a quantidade disponível.

**Solução:**

```python
estoque = 5

while estoque > 0:
    estoque -= 1 
    print(f"Venda realizada! Estoque restante: {estoque}")

print("Estoque esgotado")
```

## Exemplo 8

Ana está implementando um sistema de filtragem de livros no Buscante. A funcionalidade deve percorrer uma lista de livros e exibir o nome de cada livro disponível em estoque. No entanto, se o livro estiver esgotado, ele deve ser ignorado durante a iteração.

```python
livros = [
    {"nome": "1984", "estoque": 5},
    {"nome": "Dom Casmurro", "estoque": 0},
    {"nome": "O Pequeno Príncipe", "estoque": 3},
    {"nome": "O Hobbit", "estoque": 0},
    {"nome": "Orgulho e Preconceito", "estoque": 2}
]

for livro in livros:
    if livro["estoque"] == 0:
        continue
    print(f"Livro disponível: {livro['nome']}")
```

# Diferenças entre `while` e `for`

| **while**                  | **for**                    |
| -------------------------- | -------------------------- |
| Depende de condição lógica | Percorre uma sequência     |
| Pode virar loop infinito   | Mais seguro                |
| Usado em validações        | Usado em listas e coleções |

---
# Boas Práticas com Laços de Repetição

- Evite loops infinitos
- Prefira `for` quando possível
- Use nomes de variáveis claros
- Evite `break` excessivo
- Mantenha o código legível

---
# Erros comuns

❌ Esquecer de atualizar a variável no `while`
❌ Confundir `break` com `continue`
❌ Usar `range()` esperando que inclua o valor final

---
# Conclusão

Estruturas de repetição são essenciais para escrever programas eficientes e automatizados. O domínio de `while`, `for` e dos comandos auxiliares permite criar soluções robustas e escaláveis.
