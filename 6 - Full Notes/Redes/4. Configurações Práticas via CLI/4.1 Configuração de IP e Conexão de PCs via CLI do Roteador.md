# Visão Geral

A configuração de IP e conexões entre dispositivos por meio da CLI (*Command Line Interface*) de um roteador envolve:

1. Atribuir endereços IP às interfaces do roteador
2. Conectar PCs ao roteador por cabos apropriados
3. Configurar IPs nos PCs
4. Testar Conectividade

----
# Etapas da Configuração no CLI do Roteador

## 1. Acessar o CLI do roteador

Ao iniciar o roteador, você verá um prompt como:

```shell
Router>
```

Digite:

```bash
enable
```

Para entrar no modo privilegiado:

```bash
Router#
```

Depois entre no modo de configuração global:

```bash
configure terminal
```

Agora:

```scss
Router(config)#
```

## 2. Configurar uma interface de rede

Supondo que o PC esteja conectado à interface <font color="#4bacc6"><strong>FastEthernt0/0</strong></font>:

```bash
interface FastEthernet0/0
ip address 192.168.1.1 255.255.255.0
no shutdown
```

Caso exista outro PC em <font color="#4bacc6"><strong>FastEthernt0/1</strong></font>:

```bash
interface FastEthernet0/1
ip address 192.168.2.1 255.255.255.0
no shutdown
```

> O comando `no shutdown` ative a interface.

## 3. Verificar interfaces configuradas

Para verificar as interfaces e seus status:

```bash
exit
show ip interface brief
```

Saída esperada:

```nginx
Interface          IP-Address      OK? Method Status      Protocol
FastEthernet0/0    192.168.1.1     YES manual up          up
FastEthernet0/1    192.168.2.1     YES manual up          up
```

## 4. Conectar PCs ao roteador

No Packet Tracer ou fisicamente:

- Use **cabo direto *(straight-through)*** para conectar os PC à porta do roteador
- Conecte a <font color="#ffff00"><strong>porta Ethernet do PC</strong></font> à <font color="#00b050"><strong>interface FastEthernet do roteador</strong></font>

## 5. Configurar IP no PC

Nos PCS (via interface gráfica ou CLI):

### **Em modo gráfico (no Packet Tracer)**

1. Clique no PC > Desktop > IP Configuration
2. Defina
	- IP: `192.168.1.2`
	- Máscara: `255.255.255.0`
	- Gateway: `192.168.1.1`

### **Em CLI do PC (Packet Tracer)**

```bash
ip 192.168.1.2 255.255.255.0 192.168.1.1
```

## 6. Testar conectividade com `ping`

### No PC:

```bash
ping 192.168.1.1
```

### No roteador

```bash
ping 192.168.1.2
```

### Resposta esperada

```python
Reply from 192.168.1.2: bytes=32 time<1ms TTL=128
```

# Exemplo completo de Script (para dois PCs)

```bash
enable
configure terminal

interface FastEthernet0/0
ip address 192.168.1.1 255.255.255.0
no shutdown

interface FastEthernet0/1
ip address 192.168.2.1 255.255.255.0
no shutdown

exit
show ip interface brief
```


| **PC** | IP          | **Mácara**    | **Gateway** |
| ------ | ----------- | ------------- | ----------- |
| PC1    | 192.168.1.2 | 255.255.255.0 | 192.168.1.1 |
| PC2    | 192.168.2.2 | 255.255.255.0 | 192.168.2.1 |

> [!important] Importante
> Para comunicação entre as redes (192.168.1.0 e 192.168.2.0), o roteador funciona como **gateway**

# Dicas Finais

- Sempre use o comando `no shutdown` após configurar IP em interfaces
- Use `show running-config` para ver a configuração ativa
- Configure `hostname`, senhas e segurança se necessário
- Use `write` ou `copy running config startup-config` para salvar
