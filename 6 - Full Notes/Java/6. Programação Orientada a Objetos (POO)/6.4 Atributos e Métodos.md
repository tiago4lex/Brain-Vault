# Introdução

Dentro da **POO** em Java, as **classes** são compostas por dois elementos principais:

- **Atributos** → características (dados da classe).
- **Métodos** → comportamentos (ações que a classe pode realizar).

Esses dois elementos são a base da modelagem em Java.

---
# O que são Atributos?

Os **atributos são as variáveis dentro de uma classe**.
Eles representam as **propriedades** ou **características** de um objeto.

**Exemplo de atributo na classe `Carro`:**

```java
public class Carro {
	// Atributos (características)
	String marca;
	String modelo;
	int ano;
	double velocidade;
}
```

Aqui `marca`, `modelo`, `ano` e `velocidade` são atributos. Cada objeto **Carro** terá seus próprios valores para esses atributos.

---
# O que são Métodos?

Os **métodos** são **funções declaradas dentro da classe**.
Eles representam os **comportamentos** de um objeto.

```java
public class Carro {
	String marca;
	String modelo;
	int ano;
	double velocidade = 0;
	
	// Método acelerar
	void acelerar() {
		velocidade += 10;
		System.out.println("O carro acelerou. Velocidade atual: " + velocidade + "km/h");
	}
	
	// Método para frear
	void frear() {
		velocidade -= 10;
		if (velocidade < 0) velocidade = 0;
		System.out.prinln("O carro freou. velocidade atual: " + velocidade + "km/h");
	}
}
```

Aqui `acelerar()` e `frear()` são **métodos** que manipulam o atributo `velocidade`.

---
# Exemplo Completo: Criando Objetos com Atributos e Métodos

```java
public class Main {
	public static void main(String[] args) {
		// Criando um objeto da classe carro
		Carro carro1 = new Carro();
		carro1.marca = "Toyota";
		carro1.modelo = "Corolla";
		carro1.ano = 2020;
		
		// Usando métodos
		carro1.acelerar();
		carro1.acelerar();
		carro1.frear();
	}
}
```

**Saída:**

```text
O carro acelerou. Velocidade atual: 10.0 km/h
O carro acelerou. Velocidade atual: 20.0 km/h
O carro freou. Velocidade atual: 10.0 km/h
```

# Tipos de Métodos

## 1. Método sem retorno `void`

Apenas executa uma ação.

```java
void apresentar() {
	System.out.println("Olá, sou um carro da marca " + marca);
}
```

## 2. Método com retorno

Retorna um valor após a execução.

```java
double obterVelocidade() {
	return velocidade;
}
```

## 3. Método com parâmetros

Recebe valores para executar sua lógica.

```java
void definirVelocidade(double novaVelocidade) {
	velocidade = novaVelocidade;
}
```

# Exemplo Prático

## Classe `Aluno`

```java
public class Aluno {
	// Atributos
	String nome;
	int idade;
	double nota;
	
	// Método que apresenta o aluno
	void apresentar() {
		System.out.println("Olá, meuu nome é " + nome + " e tenho " + idade + " anos.");
	}
	
	// Método que verifica se o aluno está aprovado
	boolean aprovado() {
		return nota >= 6.0;
	}
}
```

## Criando Objetos

```java
public class Main{
	public static void main(String[] args) {
		Aluno aluno1 = new Aluno();
		aluno1.nome = "Maria";
		aluno1.idade = 23;
		aluno1.nota = 7.5;
		
		aluno1.apresentar();
		System.out.prinln("Está aprovado? " + aluno1.aprovado());
	}
}
```

**Saída:**

```text
Olá, meu nome é Maria e tenho 23 anos.
Está aprovado? true
```

# Diferença entre Atributos e Métodos

|**Atributos**|**Métodos**|
|---|---|
|Variáveis dentro da classe.|Funções dentro da classe.|
|Representam características (ex: nome, idade, velocidade).|Representam comportamentos (ex: correr, estudar, frear).|
|Guardam dados.|Manipulam os dados e definem ações.|

---
# Boas Práticas

- Sempre dar **nomes significativos** a atributos e métodos.
- Métodos geralmente usam **verbos** (ex: `acelerar()`, `calcularMedia()`, `enviarEmail()`).
- Atributos usam **substantivos** (ex: `nome`, `idade`, `saldo`).
- Mais adiante, recomenda-se **encapsular** atributos (torná-los privados) e acessá-los por métodos `get` e `set`.

---
# Conclusão

- **Atributos** = variáveis que armazenam características do objeto.    
- **Métodos** = funções que definem ações do objeto.
- Juntos, eles formam a base da **POO em Java**, permitindo modelar o mundo real dentro do código.
