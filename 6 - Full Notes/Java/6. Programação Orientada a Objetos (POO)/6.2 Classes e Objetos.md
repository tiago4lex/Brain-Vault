# Introdução

A **Programação Orientada a Objetos (POO)** é um dos pilares da linguagem Java. Dois conceitos fundamentais para entender essa forma de programar são: **Classes** e **Objetos**.

---
# Oque é uma Classe?

Uma **classe** é como uma **fôrma** ou um **molde**.

Ela define **como algo de ver** (atributos) e **o que ela pode fazer** (métodos).

- **Atributos** → características ou propriedades.
- **Métods** → comportamentos ou ações.

>[!note] Nota:
>Em Java, criamos uma classe com a palavra-chave `class`.

## Exemplo de Classe

```java
public class Carro {
	// Atributos
	String marca;
	String modelo;
	int ano;
	double velocidade;
	
	// Métodos
	void acelerar() {
		velocidade +=10;
		System.out.println("O carro acelerou. Velocidade atual: " + velocidade + " km/h);
	}
	
	void frear() {
		velocidade -= 10;
		if (velocidade < 0) velocidade = 0;
		System.out.println("O carro freaou. Velocidade atual: " + velocidade + " Km/h);
	}
}
```

Aqui temos a **classe Carro**:
- Os **atributos** são: `marca`, `modelo`, `ano` e `velocidade`.
- Os **métodos** são: `acelerar()`, `frear()`.

---
# O que é um Objeto?

Um **objeto** é a **instância de uma classe**.

Ou seja, é quando pegamos o "molde" (classe) e **criamos algo real** a partir dele.

>[!nota] Nota:
>Para criar um objeto, usamos a palavra-chave `new`.

## Exemplo de Objeto

```java
public class Main {
	public static void main(String[] args) {
		// Criando objetos a partir da classe Carro
		Carro carro1 = new Carro();
		Carro carro2 = new Carro();
		
		// Definindo valores para os atributos
		carro1.marca = "Toyota";
		carro1.modelo = "Corolla";
		carro1.ano = 2020;
		carro.velocidade = 0;
		
		carro2.marca = "Honda";
		carro2.modelo = "Civic";
		carro2.ano = 2019;
		carro2.velocidade = 0;
		
		// Usando métodos dos objetos
		carro1.acelerar();
		carro1.acelerar();
		carro1.frear();
		
		carro2.acelerar();
	}
}
```

Nesse exemplo:
- `carro1` e `carro2` são **objetos da classe Carro**.
- Cada objeto tem seus próprios **atributos independentes**.
- Quando chamamos `carro1.acelerar()`, apenas `carro1` acelera, o `carro2` continua parado.

---
# Diferença entre Classe e Objeto

| **Classe**                  | **Objeto**                                   |
| --------------------------- | -------------------------------------------- |
| É o **molde**.              | É a **coisa real criada** a partir do molde. |
| Define atributos e métodos. | Possui valores concretos nos atributos.      |
| Não ocupa memória sozinha   | Ocupa espaço na memória.                     |

**Exemplo prático:**

- Classe → "Carro" (conceito)
- Objeto → "Meu carro Toyota Corolla 2020, cor prata"

---
# Usos  e Casos Comuns

- **Modelagem do mundo real:** Classes representam entidades do dia a dia. Exemplo: `Aluno`, `ContaBancaria`, `Livro`.
- **Organização do código:** Facilita a manutenção e reutilização.
- **Reutilização:** Criar várias instâncias (objetos) a partir de um único molde.

# Outro Exemplo: Classe `Aluno`

```java
public class Aluno {
	String nome;
	int idade;
	double nota;
	
	void apresentar() {
	System.out.println("Olá, meu nome é " + nome + " e tenho " + idade + " anos.");
	}
	
	boolean aprovado() {
		return nota >= 6.0;
	}
}
```

```java
public class Main {
	public static void main(String[] args) {
		Aluno a1 = new Aluno();
		a1.nome = "Maria";
		a1.idade = "23";
		a1.nota = "7.5";
		
		a1. apresentar();
		System.out.println("Está aprovado? " + a1.aprovado());
	}
}
```

**Saída:**

```text
Olá, meu nome é Maria e tenho 23 anos.
Está aprovado? true
```

---
# Conclusão

- **Classe:** é o molde, define características e comportamentos.
- **Objeto:** é a instância da classe, algo real e concreto.
- Cada objeto criada a partir de uma classe pode ter valores diferentes, mas compartilha os métodos definidos.
