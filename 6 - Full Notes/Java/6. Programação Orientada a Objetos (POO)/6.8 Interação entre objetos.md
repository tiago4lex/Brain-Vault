# Interação entre Objetos

## O que é?

Na POO, os objetos **trabalham juntos**.  
Um objeto pode **conter outro objeto** ou **usar métodos de outro objeto**.

Essa comunicação entre classes é chamada de **interação entre objetos**.

## Exemplo: Um `Aluno` tem um `Curso`

```java
class Curso {
    String nome;

    Curso(String nome) {
        this.nome = nome;
    }

    void exibirCurso() {
        System.out.println("Curso: " + nome);
    }
}

class Aluno {
    String nome;
    Curso curso;  // relação: Aluno possui um Curso

    Aluno(String nome, Curso curso) {
        this.nome = nome;
        this.curso = curso;
    }

    void exibirDados() {
        System.out.println("Aluno: " + nome);
        curso.exibirCurso();
    }
}

public class Main {
    public static void main(String[] args) {
        Curso c1 = new Curso("Engenharia de Software");
        Aluno a1 = new Aluno("Maria", c1);

        a1.exibirDados();
    }
}
```

**Saída:**

```text
Aluno: Maria
Curso: Engenharia de Software
```

Aqui, o objeto `Aluno` **interage** com o objeto `Curso`, chamando seus métodos e acessando seus dados.

## Quando usar

- Quando uma classe **depende de outra** (ex: um carro tem um motor).
- Quando há **associação, composição ou agregação** entre objetos.
