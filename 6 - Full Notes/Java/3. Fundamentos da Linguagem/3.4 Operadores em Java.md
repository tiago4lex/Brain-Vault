# Operadores Aritméticos

Usados para realizar operações matemáticas básicas entre valores numéricos (`int`, `double`, `float`, `long`, etc.).

| Operador | Descrição                 | Exemplo  | Resultado |
| -------- | ------------------------- | -------- | --------- |
| `+`      | Soma                      | `5 + 3`  | `8`       |
| `-`      | Subtração                 | `5 - 3`  | `2`       |
| `*`      | Multiplicação             | `5 * 3`  | `15`      |
| `/`      | Divisão                   | `10 / 2` | `5`       |
| `%`      | Resto da divisão (módulo) | `10 % 3` | `1`       |

**Exemplo em código:**

```java
public class ExemploAritmetico {
	public static void main(String[] args) {
		int a = 10, b = 3;
		
		System.out.println("Soma: " + (a + b));
		System.out.println("Subtração: " + (a - b));
		System.out.println("Multiplicação: " + (a * b));
		System.out.println("Divisão: " + (a / b)); // resultado inteiro: 3
		System.out.println("Resto: " + (a % b));
	}
}
```

**Observações**

- Se ambos os operandos forem inteiros, a divisão também será inteira (descarta as casas decimais).
- Para divisão com decimais, é necessário usar pelo menos um operando `double` ou `float`: `System.out.println(10.0 / 3)`, resultado: `3.333...`

# Operadores Relacionais

Usados para comprar dois valores e retornam um resultado **booleano** (`true` ou `false`).

| Operador | Descrição        | Exemplo  | Resultado |
| -------- | ---------------- | -------- | --------- |
| ==       | Igual a          | `5 == 5` | `true`    |
| `!=`     | Diferente de     | `5 != 3` | `true`    |
| `>`      | Maior que        | `7 > 3`  | `true`    |
| `<`      | Menor que        | `2 < 5`  | `true`    |
| `>=`     | Maior ou igual a | `5 >= 5` | `true`    |
| `<=`     | Menor ou igual a | `3 <= 2` | `false`   |

**Exemplo em código:**

```java
public class ExemploRelacional {
	public static void main(String[] args) {
		int x = 10, y = 20;
		System.out.println(x == y);  // false
        System.out.println(x != y);  // true
        System.out.println(x < y);   // true
        System.out.println(x >= 10); // true
	}
}
```

**Importante:**

- Para comprar **objetos** (como `String`) é preciso usar `equals()` e não == :

```java
String a = "Java";
String b = "Java";
System.out.println(a == b);        // pode dar false em alguns casos
System.out.println(a.equals(b));   // true (correto para conteúdo)
```

# Operadores Lógicos

Usados para combinar expressões booleanas. O resultado também é um **booleano** (`true` ou `false`).

|Operador|Nome|Significado|Exemplo|Resultado esperado|
|---|---|---|---|---|
|`&&`|E lógico (AND)|Verdadeiro se **ambos** forem verdadeiros|`(x > 5 && y < 10)`|`true` se `x > 5` **e** `y < 10`|
|`\|`|OU lógico (OR)|Verdadeiro se **pelo menos um** for verdadeiro|`(x > 5 \| y < 10)`|`true` se **uma ou ambas** forem verdadeiras|
|`!`|NÃO lógico (NOT)|Inverte o valor lógico|`!(x > 5)`|`false` se `x > 5`|

**Exemplo em código:**

```java
public class ExemploLogico {
    public static void main(String[] args) {
        int idade = 20;
        boolean temCarteira = true;

        if (idade >= 18 && temCarteira) {
            System.out.println("Pode dirigir.");
        }

        boolean aprovado = false;
        System.out.println(!aprovado); // true
    }
}
```

**Observação:**

- `&&` e `||` fazem **avaliação de curto-circuito**: se a primeira condição já determina o resultado, a segundo nem é avaliada.

# Operadores de Atribuição

Usados para atribuir **valores a variáveis**. O operador = é o maios básico, e os demais combinam operações aritméticas com atribuição.

| Operador | Exemplo  | Equivalente a | Resultado            |
| -------- | -------- | ------------- | -------------------- |
| `=`      | `x = 5`  | —             | `x` recebe `5`       |
| `+=`     | `x += 5` | `x = x + 5`   | soma e atribui       |
| `-=`     | `x -= 5` | `x = x - 5`   | subtrai e atribui    |
| `*=`     | `x *= 5` | `x = x * 5`   | multiplica e atribui |
| `/=`     | `x /= 5` | `x = x / 5`   | divide e atribui     |
| `%=`     | `x %= 5` | `x = x % 5`   | resto e atribui      |

**Exemplo em código:**

```java
public class ExemploAtribuicao {
	public static void main(String[] args) {
		int x = 10;
		x += 5; // x = 15
		x *= 2; // x = 30
		x /= 3; // x = 10
		
		System.out.println(x);
	}
}
```

# Operadores de Incremento e Decremento

Usados para **aumentar ou diminuir 1 unidade** no valor de uma variável numérica. Podem ser **prefixados** ou **pósfixados**, com diferenças sutis.

| Operador | Significado | Exemplo        | Resultado        |
| -------- | ----------- | -------------- | ---------------- |
| `++`     | Incremento  | `x++` ou `++x` | Soma 1 a `x`     |
| `--`     | Decremento  | `x--` ou `--x` | Subtrai 1 de `x` |

**Diferença entre pré e pós**:

- `++x` **(pré-incremento)** → incrementa e depois usa o valor.
- `x++` **(pós-incremento)** → usa o valor atual e depois incrementa.

**Exemplo em código:**

```java
public class ExemploIncremento {
    public static void main(String[] args) {
        int x = 5;
        System.out.println(x++); // imprime 5, depois x vira 6
        System.out.println(++x); // x vira 7, imprime 7

        int y = 10;
        System.out.println(y--); // imprime 10, depois y vira 9
        System.out.println(--y); // y vira 8, imprime 8
    }
}

```

# Boas Práticas

- Use parênteses para deixar expressões complexas mais claras:  
    `if ((x > 5 && y < 10) || z == 0)`
    
- Tome cuidado com tipos diferentes (ex.: `int` e `double`): faça **casting** se necessário.
- Use `equals()` para comparar conteúdo de objetos (`String`, etc.).
- Lembre-se que `x++` e `++x` têm efeitos diferentes dentro de expressões.
