# Introdução

A programação em Java exige que muitas vezes seja feita **entrada de dados** (informações fornecidas pelo usuário) e **saída de dados** (exibição de resultados ou mensagens na tela).

Para isso, os recursos mais usados são:

- A classe `Scanner` (entrada).
- O objeto `System.out` (saída).

---
# Entrada de Dados com `Scanner`

A classe `Scanner` está presente no pacote `java.util` e é amplamente usada para capturar dados digitados pelo usuário no console.

## Com usar `Scanner`?

Antes de usar, é necessário importar a classe:

```java
import java.util.Scanner;
```

Em seguida, criamos um objeto `Scanner` associado à entrada padrão (teclado):

```java
Scanner sc = new Scanner(System.in);
```

## Métodos do `Scanner`

Cada método captura um tipo específico de dado.

### Leitura de String (até espaço em branco)

```java
System.out.println("Digite o seu nome: ");
String nome = sc.next();

System.out.println("Nome: " + nome);
```

## Leitura de String completa (com espaços)

```java
System.out.println("Digite seu nome completo: ");
String nomeCompleto = sc.nextLine();

System.out.println("Nome compleo: " + nomeCompleto);
```

### Leitura de números inteiros

```java
System.out.println("Digite sua idade: ");
int idade = sc.nextInt();

System.out.println("Idade: " + idade);
```

### Leitura de números decimais

```java
System.out.print("Digite seu salário: ");
double salario = sc.nextDouble();

System.out.println("Salário: " + salario).
```

### Leitura de caracteres

- Não existe método direto, mas pode ser feito da seguinte maneira:

```java
System.out.println("Digite uma letra: ");
char letra = sc.next().charAt(0);

System.out.println("Letra: " + letra);
```

## Regras Importantes do `Scanner`

1. Sempre importar com `import java.util.Scanner;`.
2. O `Scanner` diferencia números decimais com **ponto(`.`)**, não com vírgula. Exemplo: `10.5`.
3. O método `nextLine()` pode ser afetado por leituras anteriores com `nextInt()` ou `nextDouble()`. Nesse caso, é comum precisar de um `sc.nextLine()` extra para consumir a quebra de linha.
4. É uma boa prática fechar o `Scanner` ao final:

```java
sc.close();
```

# Saída de Dados com `System.out`

O objeto `System.out` é responsável pela **saída de dados no console**. Ele possui alguns métodos principais:

## `System.out.print()`

Imprime uma mensagem **sem pular linha** no final.

```java
System.out.print("Olá ");
System.out.print("Mundo!");
```

**Saída:**

```
Olá Mundo!
```

## `System.out.println()`

Imprime uma mensagem e **adiciona uma quebra de linha** no final.

```java
System.out.println("Olá");
System.out.println("Mundo!");
```

**Saída:**

```
Olá
Mundo!
```

## `System.out.printf()`

Permite imprimir valores formatados, semelhante ao `printf` de C.

```java
double preco = 10.5
System.out.printf("O preço é: R$.2f%n", preco);
```

**Saída:**

```
O preço é: R$10.50
```

**Marcadores mais usados no `printf`:**

- `%d` → inteiro
- `%f` → float
- `%.2f` → float com 2 casas decimais
- `%s` → string
- `%n` → quebra de linha (melhor que `\n`, pois é portátil entre sistemas).

## Exemplo completo com `Scanner` e `System.out`

```java
import java.util.Scanner;

public class EntradaSaida {
	public staticc void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		
		System.out.print("Digite seu nome: ");
		String nome = sc.nextLine();
		
		System.out.print("Digite sua idade: ");
		int idade = sc.nextInt();
		
		System.out.print("Digite seu slaário: ");
		double salario = sc.nextDouble();
		
		System.out.printf("%nOlá %s!%n", nome);
		System.out.printf("Você tem %d anos e recebe R$%.2f%n", idade, salario);
		
		sc.close();
	}
}
```

**Saída:**

```
Digite seu nome: Maria
Digite sua idade: 22
Digite o seu salário: 4320

Olá Maria!
Você tem 22 anos e recebe R$4320,00
```

# Comparação rápida

|Entrada/Saída|Método|Observação|
|---|---|---|
|Entrada|`next()`|Lê até espaço em branco.|
|Entrada|`nextLine()`|Lê a linha inteira.|
|Entrada|`nextInt()`|Lê inteiro.|
|Entrada|`nextDouble()`|Lê número decimal.|
|Saída|`print()`|Imprime sem quebra de linha.|
|Saída|`println()`|Imprime com quebra de linha.|
|Saída|`printf()`|Imprime formatado.|

---
# Resumo

- Use `Scanner` para **capturar dados do usuário**.
    
- Use `System.out.print/println/printf` para **exibir mensagens e resultados**.
    
- Entender a diferença entre métodos é essencial para evitar erros comuns, como quebra de linha não lida.
