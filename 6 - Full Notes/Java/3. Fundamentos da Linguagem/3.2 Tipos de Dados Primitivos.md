# Introdução

O Java é uma linguagem **fortemente tipada**, ou seja, cada variável precisa ter um tipo definido no momento de sua declaração. Esses tipos pode ser:

- **Primitivos** → tipos básicos fornecidos pela linguagem.
- **Referência** → objetos (como `String`, classes, arrays).

---
# Conceito de tipos primitivos

- São os **blocos básicos de construção** da linguagem.
- Representam valores simples (números, caracteres, booleanos).
- **Não são objetos**, mas possuem classes wrapper correspondentes (ex.: `int` ↔ `Integer`, `double` ↔ `Double`).
- São armazenados diretamente na memória stack, o que os torna mais rápidos e eficientes.

---
# Lista dos 8 tipos primitivos

| Tipo      | Tamanho                                  | Faixa de valores                                   | Valor padrão | Classe Wrapper |
| --------- | ---------------------------------------- | -------------------------------------------------- | ------------ | -------------- |
| `byte`    | 8 bits                                   | -128 a 127                                         | 0            | `Byte`         |
| `short`   | 16 bits                                  | -32.768 a 32.767                                   | 0            | `Short`        |
| `int`     | 32 bits                                  | -2.147.483.648 a 2.147.483.647                     | 0            | `Integer`      |
| `long`    | 64 bits                                  | -9,22e18 a 9,22e18                                 | 0L           | `Long`         |
| `float`   | 32 bits                                  | ±3,40282347E+38 (7 dígitos de precisão)            | 0.0f         | `Float`        |
| `double`  | 64 bits                                  | ±1,79769313486231570E+308 (15 dígitos de precisão) | 0.0d         | `Double`       |
| `char`    | 16 bits                                  | 0 a 65.535 (caracteres Unicode)                    | '\u0000'     | `Character`    |
| `boolean` | 1 bit (representado como `true`/`false`) | `true` ou `false`                                  | `false`      | `Boolean`      |

# Tipos Inteiros

## `byte`

- Uso: útil para economia de memória em arrays grandes, arquivos binários e controle de fluxo baixo nível.

**Exemplo:**

```java
byte idade = 25;
System.out.println("idade: " + idade);
```

## `short`

- Uso: **indicado para variáveis inteires menores que `int`**, mas maiores que `byte`.

**Exemplo:**

```java
short populacaoCidade = 32000;
System.out.println("População: " + populacaoCidade);
```

## `int`

- Tipo inteiro mais usado em Java.
- Ideal para contadores, índices de arrays e cálculos que não ultrapassam sua faixa.

**Exemplo:**

```java
int ano = 2025;
System.out.println("Ano: " + ano);
```

## `long`

- **Uso:** para números inteiros muito grandes.
- Necessário usar `L` no final do número.

**Exemplo:**

```java
long distancia = 15000000000L; // 15 bilhões
System.out.println("Distância: " + distancia);
```

---
# Tipos de ponto flutuante

## `float`

- Armazena números decimais com **precisão simples** (32 bits).
- Precisa do sufixo `f` ou `F`.
- Uso: **aplicações que não exigem grande precisão** (ex: gráficos, jogos).

**Exemplo:**

```java
float nota = 8.75f;
System.out.println("Nota: " + nota);
```

## `double`

- Armazena números decimais com **precisão dupla (64 bits)**.
- É o tipo **padrão** para valores decimais em Java.
- Uso: cálculos matemáticos, financeiros e científicos.

**Exemplo:**

```java
double preco = 1999.99
System.out.println("Preço: R$" + preco);
```

---
# Tipo caractere

## `char`

- Representa um único caractere Unicode (16 bits).
- Pode armazenar letras, dígitos, símbolos e até emojis.

**Exemplo:**

```java
char letra = 'A';
char simblo = '#';
char unicode = '\u2764'; // ❤
System.out.println(letra + " " + simbolo + " " + unicode);
```

---
# Tipo lógico

## `boolean`

- Representa valores lógicos `true` ou `false`.
- Muito usado em estruturas condicionais e de repetição.

**Exemplo:**

```java
boolean ativo = true;

if (ativo) {
	System.out.println("Usuário está ativo!");
}
```

---
# Casos de uso

- **`byte` e `short`** → economizar memória em grandes coleções.
- **`int`** → contadores, índices, cálculos em geral.
- **`long`** → números grandes, como cálculos financeiros ou astronômicos.
- **`float`** → aplicações gráficas, quando não é necessária muita precisão.
- **`double`** → cálculos matemáticos e científicos.
- **`char`** → representar letras, símbolos e caracteres Unicode.
- **`boolean`** → controle de fluxo (condições e loops).
