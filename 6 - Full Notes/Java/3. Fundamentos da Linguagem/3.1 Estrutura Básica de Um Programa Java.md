# Introdução

Um programa Java segue uma organização bem definida. Essa estrutura é composta por **pacote (opcional), importações, definição da classe e métodos**, sendo o ponto de partida o **método `main`**.

---
# Estrutura Geral

```java
// Declaração de pacote (opcional)
package nomeDoPacote;

// Importação de bibliotecas (opcional)
import java.util.Scanner;

// Definição da classe principal
public class MeuPrograma {
	
	// Método principal: ponto de entrada da aplicação
	public static void main(String[] args) {
		// Código do programa
		System.out.println("Olá, mundo!");
	}
}
```

---
# Componentes Principais

## Pacote (`package`)

- É a forma de organizar classes em Java.
- Ajuda a manter o código modular e organizado.
- Opcional em programas pequenos, mas comum em projetos grandes.

## Importações (`import`)

- Permite usar classes de outras bibliotecas ou pacotes.
- Algumas classes do pacote `java.lang` já são importadas automaticamente (como `String`, `Math`).

**Exemplo:**

```java
import java.util.Scanner;
```

## Classe principal (`class`)

- Todo programa Java deve ter ao menos **uma classe pública** com o **mesmo nodo do arquivo**.
- Dentro da classe ficam os atributos (variáveis) e métodos (ações).

**Exemplo:**

```java
public class MeuPrograma {
	// atributos e métodos
}
```

Se o arquivo se chama `MeuPrograma.java` a classe pública deve se chamar `MeuPrograma`.

## Método principal (`main`)

- É o ponto de entrada do programa.
- Sempre deve ter essa assinatura:

```java
public static void main(String[] args)
```

- `public`: acessível de qualquer lugar.
- `static`: pode ser executado sem precisar instanciar a classe.
- `void`: não retorna valor.
- `String[] args`: permite passar argumentos pela linha de comando.

**Exemplo simples:**

```java
public static void main(String[] args) {
	System.out.println("Executando programa...");
}
```

# Estrutura interna da classe

Uma classe pode conter:

1. **Atributos** (variáveis de instância ou classe)
2. **Construtores** (opcionais, para inicialização)
3. **Métodos** (ações)

**Exemplo:**

```java
public class Carro {
	// Atributos
	String cor;
	double velocidadeMaxima;
	
	// Construtor
	public Carr(String cor, double velocidadeMaxima) {
		this.cor = cor;
		this.velocidadeMaxima = velocidadeMaxima;
	}
	
	// Método
	public void acelerar() {
		System.out.println("O carr está acelerando!")
	}
}
```

# Exemplo completo

```java
package aplicacao;

import java.util.Scanner;

public class ProgramaExemplo {
	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		
		// Entrada de dados
		System.out.println("Digite seu nome: ");
		String nome = sc.nextLine();
		
		// Processamento
		String mensagem = "Olá, " + nome + "! Seja bem-vindo ao Java";
		
		// Saída de dados
		System.out.println(mensagem);
		
		sc.close();
	}
}
```

# Fluxo básico de execução Java

1. **Escrevemos  o código-fonte** no arquivo `.java`.
2. **Compilamos** usando `javac` (gera arquivo `.class` em bytecode).
3. **Executamos** com `java` (a JVM interpreta o bytecode).

**Exemplo no terminal:**

```bash
javac ProgramaExemplo.java
java aplicacao.ProgramaExemplo
```

---
# Resumindo

- Um programa Java precisa ter ao menos **uma classe pública** com o mesmo nome do arquivo.
- O método `main` é o ponto de entrada.
- Opcionalmente, podemos usar **pacotes e importações** para organizar o código.
- A estrutura básica sempre segue o mesmo padrão, o que facilita a organização de projetos maiores.
