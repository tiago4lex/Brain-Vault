# Introdução

As **estruturas de repetição** (ou **loops**) permitem executar um bloco de código várias vezes de forma controlada. Em Java, existem diferentes formas de repetição, cada uma adequada para um tipo de situação.

---
# Estrutura `While`

A instrução `while` repete um bloco de código enquanto a condição especificada for verdadeira.

## Sintaxe

```java
while (condição) {
	// bloco de código
}
```

## Exemplo

```java
int contador = 1;

while (contador <= 5) {
	System.out.println("Contagem: " + contador);
	contador++;
}
```

**Saída:**

```
Contagem: 1
Contagem: 2
Contagem: 3
Contagem: 4
Contagem: 5
```

✅ **Uso:** quando não sabemos previamente quantas vezes o loop será executado.  
⚠️ **Cuidado:** se a condição nunca for falsa, teremos um **loop infinito**.

---
# Estrutura `do-while`

O `do-while` é semelhante ao `while`, mas garante que o bloco seja executado **pelo mneos uma vez,** já que **a condição é testada após a execução**.

## Sintaxe

```java
do {
	// bloco de código
} while (condição);
```

## Exemplo

```java
int numero = 0;

do {
	System.out.println("Número: " + numero);
	numero++;
} while (numero < 3);
```

**Saída:**

```
Número: 0
Número: 1
Número: 2
```

✅ **Uso:** quando precisamos que o código seja executado ao menos uma vez, mesmo que a condição inicial seja falsa.

---
# Estrutura `for`

O `for` é usado quando sabemos previamente o número de repetições. Ele é composto por três partes:

1. **Inicialização** (executada uma vez antes do loop começar).
2. **Condição** (testada a cada iteração).
3. **Incremento/decremento** (executado ao final de cada iteração).

## Sintaxe

```java
for (inicialização; condição; incremento) {
	// bloco de código
}
```

## Exemplo

```java
for (int i = 1; i <= 5; i++) {
	System.out.println("Valor de i: " + i);
}
```

**Saída:**

```
Valor de i: 1
Valor de i: 2
Valor de i: 3
Valor de i: 4
Valor de i: 5
```

✅ **Uso:** ideal quando sabemos quantas vezes o loop deve rodar.

---
# Estrutura `for-each` *(enhanced for)*

O `for-each` é uma versão simplificado do `for`, usada para percorrer coleções, arrays ou listas.

## Sintaxe

```java
for (tipo variave: coleção) {
	// bloco de código
}
```

## Exemplo

```java
String[] nomes = {"Ana", "Bruno", "Carlos"};

for (String nome : nomes) {
	System.out.println("Nome: " + nome);
}
```

**Saída:**

```
Nome: Ana
Nome: Bruno
Nome: Carlos
```

✅ **Uso:** facilita a leitura quando precisamos percorrer todos os elementos de um array ou lista.  
⚠️ Não permite modificar o índice ou acessar elementos fora da sequência direta.

# Controle de fluxo em laços

Além das estruturas principais, existem comandos que alteram o comportamento dos loops.

## `break`

Interrompe a execução do loop imediatamente.

```java
for (int i = 1; i <=10; i++) {
	if (i == 5) {
		break; // interrompe o laço
	}
	System.out.println("i = " + i);
}
```

**Saída:**

```
i = 1
i = 2
i = 3
i = 4
```

## `continue`

Pula a iteração atual e vai direto para a próxima

```java
for (int i = i; i <= 5; i++) {
	if (i == 3) {
		continue; // pulla o número 3
	}
	System.out.prinln("i = " + i);
}
```

**Saída:**

```
i = 1
i = 2
i = 4
i = 5
```

---
# Comparação entre os laços

|Estrutura|Quando usar|
|---|---|
|`while`|Quando não sabemos quantas vezes o loop deve rodar (condição depende de algo externo).|
|`do-while`|Quando precisamos executar pelo menos uma vez, mesmo sem certeza da condição.|
|`for`|Quando sabemos o número de repetições ou precisamos de um contador.|
|`for-each`|Quando queremos percorrer todos os elementos de uma coleção ou array.|

# Exemplo prático combinando estruturas

```java
public class EstruturasRepeticao {
	public static void main(String[] args) {
		// for
		for (int i = 1; i <= 3; i++) {
			System.out.println("for -> i: " + i);
		}
		
		// while
		int j = 1;
		while (j <= 3) {
			System.out.println("while -> j: " + j);
			j++;
		}
		
		// do-while
		int k = 1;
		do {
			System.out.println("do-while -> k: " + k);
			k++;
		} while (k <= 3);
		
		// for-each
		String[] cores = {"Vermelho",  "Verde", "Azul"};
		for (String cor: cores) {
			System.out.println("for-each -> cor: " + cor);
		}
	}
}
```

**Saída:**

```
for -> i: 1
for -> i: 2
for -> i: 3
while -> j: 1
while -> j: 2
while -> j: 3
do-while -> k: 1
do-while -> k: 2
do-while -> k: 3
for-each -> cor: Vermelho
for-each -> cor: Verde
for-each -> cor: Azul
```

# Conclusão

- O Java oferece diversas **estruturas de repetição** para diferentes cenários.
    
- A escolha depende se sabemos ou não a quantidade de repetições e se lidamos com arrays ou listas.
    
- O uso correto de `break` e `continue` permite maior controle do fluxo.
