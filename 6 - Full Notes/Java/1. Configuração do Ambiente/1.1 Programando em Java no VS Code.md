# 1. Instalando o JDK

O JDK (Java Development Kit) é o kit de desenvolvimento necessário para compilar e executar códigos Java.

## 1.1 No Windows

1. Para instalar no Windows, podemos acessar o [site da Oracle](https://www.oracle.com/java/technologies/downloads/#jdk17-windows) ou buscar no navegador por “Java Download Oracle” e acessar o primeiro link.
2. Clique em **Latest LTS** (versão estável mais recente, por exemplo JDK 17).

![[Pasted image 20250916235219.png]]

3. Escolha o instalador para Windows (.msi).

![[Pasted image 20250916235319.png]]

4. Execute o instalador e siga os passos padrão.

![[Pasted image 20250916235334.png]]

![[Pasted image 20250916235343.png]]

![[Pasted image 20250916235350.png]]

5. Após a instalação, verifique e anote o caminho do diretório do JDK.  

```bash
java -version
javac -version

C:\Program Files\Java\jdk-17\
```

## 1.2 No Linux

1. Abra o terminal e execute:

```bash
sudo apt update
sudo apt install default-jdk -y
```

2. Para outras distribuições (como Fedora ou Arch Linux), use o gerenciado de pacotes da distro (`dnf`, `pacman`, etc.).
3. Confirme a instalação.

```bash
java -version
javac -version
```

# 2. Configurando Variáveis de Ambiente

As variáveis de ambiente permitem que o sistema reconheça os comandos `java` e `javac` de qualquer lugar no terminal.

## 2.1 No Windows

1. Pressione `Win + S` e pesquise: **variáveis de ambiente**.
2. Clique em **Editar as variáveis de ambiente do sistema**.
3. Na aba **Avançado**, clique em **Variáveis de Ambiente**.

![[Pasted image 20250917000103.png]]

4. Em **Variáveis do sistema**:
	- Clique em **Novo:**
		- Nome da variável: `JAVA_HOME`
		- Valor da variável: caminho do JDK
	- Selecione a variável `Path`, clique em **Editar** e depois em **Novo**:

![[Pasted image 20250917000123.png]]

![[Pasted image 20250917000144.png]]

```per
%JAVA_HOME%\bin
```

5. Clique em OK em todas as janelas.
6. Abra o **Prompt de Comando** e teste:

```bash
java -version
javac -version
```

## 2.2 No Linux

1. Abra o terminal e digite:

```bash
sudo update-alternatives --config java
sudo update-alternatives --config javac
```

(Caso tenha várias versões instaladas, selecione a correta).

2. Descubra o caminho do JDK:

```bash
readlink -f $(which java)
```

(vai retornar algo como `/usr/lib/jvm/java-21-openjdk-amd64/bin/java`)

3. Abra o arquivo `~/.bashrc` ou `~/.zshrc` e adicione:

```bash
export JAVA_HOME=/usr/lib/jvm/java-21-openjdk-amd64
export PATH=$JAVA_HOME/bin:$PATH
```

4. Salve e recarregue:

```bash
source ~/.bashrc
```

5. Teste:

```bash
java -version
javac -version
```

# Criando e Executando seu Primeiro Programa Java

1. Abra o VS Code.
2. Crie uma pasta de projeto e um arquivo chamado `Main.java`.
3. Escreva o seguinte código:

```java
public class Main {
	public static void main(String[] args) {
		System.out.println("Hello World!");
	}
}
```

4. Abra o terminal integrado do VS Code (`Ctrl + J`) e execute:

```bash
javac Main.java
java Main
```

Se aparecer a mensagem `Hello World!`, está tudo funcionado corretamente.

# Dicas Extras

- Sempre salve seus arquivos com o **mesmo nome da classe pública**.  
    Ex: `public class Main` → arquivo `Main.java`.

- Use `java NomeDaClasse` (sem `.java`) ao executar.
- Caso use Linux, pode ser necessário dar permissão de execução ao VS Code:

```bash
chmod +x code
```
