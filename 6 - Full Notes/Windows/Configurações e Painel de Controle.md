# Introdução

Em um sistema Windows, os principais locais para fazer alterações são o menu Configurações e o Painel de Controle.

Por muito tempo, o Painel de Controle foi o local principal para fazer alterações no sistema, como adicionar uma impressora, desinstalar um programa, etc.

O menu Configurações foi introduzido no Windows 8, o primeiro sistema operacional Windows voltado para tablets com tela sensível ao toque, e ainda está disponível no Windows 10. Aliás, o menu Configurações agora é o principal local que um usuário acessa se deseja alterar o sistema.

Ambos podem ser acessados ​​pelo Menu Iniciar.

![[Pasted image 20250630191410.png]]

O Painel de Controle é o menu onde você acessa configurações mais complexas e executa ações mais complexas. Em alguns casos, você pode começar em Configurações e terminar no Painel de Controle.

Por exemplo, em Configurações, ao clicar em **Rede e Internet**. A partir daí, clique em **Alterar opções do adaptador**.

![[Pasted image 20250630191520.png]]

Observe que a próxima janela que aparece é do Painel de Controle.

![[Pasted image 20250630191543.png]]

Se não tiver certeza de qual abrir se desejar alterar uma configuração, use o menu Iniciar e pesquise por ela.

No exemplo abaixo, a busca foi "papel de parede". Observe que poucos resultados foram retornados.

![[Pasted image 20250630191554.png]]

Se clicarmos em Melhor correspondência, uma janela para o menu Configurações aparecerá para fazer alterações no papel de parede.
