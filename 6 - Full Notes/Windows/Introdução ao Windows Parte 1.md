# Introdução

O sistema operacional Windows tem uma longa história desde 1985, e atualmente é o sistema operacional dominante tanto em redes domésticas quanto corporativas. Por isso, o Windows sempre foi alvo de hackers e criadores de malware.

---
# História

O Windows XP era uma versão popular do Windows e tinha uma longa história. A Microsoft anunciou o Windows Vista, que representou uma reformulação completa do sistema operacional Windows. Houve muitos problemas com o Windows Vista. Ele não foi bem recebido pelos usuários do Windows e foi rapidamente descontinuado.

Quando a Microsoft anunciou a data de fim de vida útil do Windows XP, muitos clientes entraram em pânico. Corporações, hospitais, etc., se apressaram e testaram a próxima versão viável do Windows, que era o Windows 7, em muitos outros hardwares e dispositivos. Os fornecedores tiveram que trabalhar contra o relógio para garantir que seus produtos funcionassem com o Windows 7 para seus clientes. Se não conseguissem, seus clientes tinham que rescindir o contrato e encontrar outro fornecedor que atualizasse seus produtos para funcionar com o Windows 7. Foi um pesadelo para muitos, e a Microsoft tomou nota disso.

![[Pasted image 20250612190357.png]]

# Diferentes Versões

O Windows 7, tão rapidamente quanto foi lançado logo depois, foi marcado com uma data de fim de suporte. O Windows 8.x chegou e saiu, e teve vida curta, como o Vista.

Então chegou o Windows 10, seguido pelo [Windows](https://www.microsoft.com/en-us/windows/windows-11?r=1) 11, que é a versão atual do sistema operacional Windows para computadores desktop.

O Windows 11 vem em duas versões: Home e Pro. É possível ver diferença entre o Home e o Pro [aqui](https://www.microsoft.com/en-us/windows/compare-windows-11-home-vs-pro-versions#tabs1-2).

Embora não tenhamos falado sobre servidores, a versão atual do sistema operacional Windows para servidores é o [Windows Server 2025](https://www.microsoft.com/en-us/windows-server/).

Muitos críticos gostam de criticar a Microsoft, mas a empresa tem feito grandes progressos para melhorar a usabilidade e a segurança a cada nova versão do Windows.

>[!note] Observação: 
>A edição do Windows para a VM conectada é o Windows Server 2019 Standard, conforme visto nas **Informações do Sistema**.

# A área de trabalho (GUI)

A Área de Trabalho do Windows, também conhecida como interface gráfica do usuário ou GUI, é a tela de boas-vindas ao efetuar login em uma máquina com Windows 10.

Tradicionalmente, você precisa passar pela tela de login primeiro. A tela de login é onde você precisa inserir credenciais de conta válidas; geralmente, um nome de usuário e senha de uma conta do Windows preexistente naquele sistema específico ou no ambiente do Active Directory (se for uma máquina associada a um domínio).

![[Pasted image 20250612190537.png]]

A captura de tela acima é um exemplo de uma área de trabalho típica do Windows. Cada componente que compõe a interface gráfica do usuário (GUI) é explicado brevemente a seguir.

1. A Área de Trabalho
2. Menu Iniciar
3. Caixa de Pesquisa (Cortana)
4. Visualização de Tarefas
5. Barra de Tarefas
6. Barras de Ferramentas
7. Área de Notificação

## A Área de Trabalho

A área de trabalho é onde você encontrará atalhos para programas, pastas, arquivos, etc. Esses ícones estarão bem organizados em pastas ordenadas alfabeticamente ou espalhados aleatoriamente, sem nenhuma organização específica na área de trabalho. Em ambos os casos, esses itens normalmente são colocados na área de trabalho para acesso rápido.

A aparência da área de trabalho pode ser alterada de acordo com sua preferência. Ao clicar com o botão direito em qualquer lugar da área de trabalho, um menu de contexto será exibido. Este menu permitirá que você altere o tamanho dos ícones da área de trabalho, especifique como deseja organizá-los, copie/cole itens na área de trabalho e crie novos itens, como uma pasta, atalho ou documento de texto.

![[Pasted image 20250612190712.png]]

Em **Configurações de vídeo**, você pode alterar a resolução e a orientação da tela. Caso tenha várias telas de computador, você pode definir as configurações para várias telas aqui.

![[Pasted image 20250612190738.png]]

>[!note] Nota:
>Em uma sessão de Área de Trabalho Remota, algumas das configurações de exibição serão desabilitadas.

![[Pasted image 20250612190859.png]]

Você também pode alterar o papel de parede selecionando **Personalizar**.

![[Pasted image 20250612191257.png]]

Em Personalizar, você pode alterar a imagem de fundo da área de trabalho, alterar fontes, temas, esquema de cores, etc.

## Menu Iniciar

Em versões anteriores do Windows, a palavra **"Iniciar"** era visível no canto inferior esquerdo da interface gráfica do usuário (GUI) da área de trabalho. Em versões modernas do Windows, como o Windows 10, a palavra "Iniciar" não aparece mais, mas sim o logotipo do Windows. Embora a aparência do Menu Iniciar tenha mudado, sua finalidade geral é a mesma.

O Menu Iniciar fornece acesso a todos os aplicativos/programas, arquivos, ferramentas utilitárias, etc., que são mais úteis.

Clicando no logotipo do Windows, o Menu Iniciar será aberto. O Menu Iniciar é dividido em seções.

![[Pasted image 20250612191345.png]]

1. Esta seção do Menu Iniciar fornece atalhos rápidos para ações que você pode realizar com sua conta ou sessão de login, como fazer alterações em sua conta de usuário, bloquear sua tela ou sair da conta. Outros atalhos específicos para sua conta são a pasta Documentos (ícone de documento) e a pasta Imagens (ícone de imagens). Por fim, o ícone de engrenagem/engrenagem levará você à tela Configurações, e o ícone de energia permitirá que você se desconecte de uma sessão da Área de Trabalho Remota, desligue o computador ou reinicie-o.

Na imagem abaixo, você pode ver o que cada um dos ícones representa. Para expandir esta seção, clique no ícone que se assemelha a um hambúrguer na parte superior.

![[Pasted image 20250612191431.png]]

2. Esta seção mostrará todos os aplicativos/programas adicionados recentemente na parte superior e todos os aplicativos/programas instalados (que estão configurados para aparecer no Menu Iniciar). Nesta seção, você também verá que os aplicativos/programas serão listados em ordem alfabética. Cada letra terá sua própria seção. Veja abaixo.

![[Pasted image 20250612191526.png]]

Na imagem acima, a primeira caixa é onde os aplicativos/programas adicionados recentemente aparecerão. A segunda caixa é onde todos os aplicativos/programas instalados aparecerão.

>[!note] Observação:
>Na máquina virtual, o Google Chrome não aparecerá mais como um programa adicionado recentemente.

Se você tiver uma lista LONGA de aplicativos/programas instalados, poderá pular para uma seção específica da lista clicando nos títulos das letras para abrir uma grade alfabética. Veja abaixo.

![[Pasted image 20250612191623.png]]

>[!note] Observação:
>As letras brancas correspondem aos títulos das letras.

3. O lado direito do Menu Iniciar é onde você encontrará ícones para aplicativos/programas ou utilitários específicos. Esses ícones são conhecidos como blocos. Alguns blocos são adicionados a esta seção por padrão. Se você clicar com o botão direito do mouse em qualquer um dos blocos, adivinhou; um menu aparecerá para permitir que você execute mais ações no bloco selecionado, como redimensionar o bloco, desfixá-lo do Menu Iniciar, visualizar suas Propriedades, etc.

## Barra de Tarefas

Alguns dos componentes estão habilitados e visíveis por padrão. A Barra de Ferramentas (6), por exemplo, foi habilitada para fins de demonstração.

Alguns desses componentes podem ser desabilitados ao clicar com o botão direito do mouse na Barra de Tarefas para abrir um menu de contexto que permitirá fazer alterações.

![[Pasted image 20250612192138.png]]

Todos os aplicativos/programas, pastas, arquivos, etc., que você abrir/iniciar aparecerão na barra de tarefas.

![[Pasted image 20250612192145.png]]

Ao passar o mouse sobre o ícone, será exibida uma miniatura de pré-visualização, juntamente com uma dica de ferramenta. Essa dica é útil se você tiver muitos aplicativos/programas abertos, como o Google Chrome, e quiser descobrir qual instância do Google Chrome é a que você precisa destacar.

Ao fechar qualquer um desses itens, eles desaparecerão da barra de tarefas (a menos que você os tenha fixado explicitamente na barra de tarefas).

## A Área de Notificação

A Área de Notificação, normalmente localizada no canto inferior direito da tela do Windows, é onde a data e a hora são exibidas. Outros ícones possivelmente visíveis nessa área são o ícone de volume e o ícone de rede/sem fio, para citar alguns. Os ícones podem ser adicionados ou removidos da Área de Notificação nas configurações da Barra de Tarefas.

![[Pasted image 20250612192236.png]]

A partir daí, role para baixo até a seção Área de Notificação para fazer alterações.
