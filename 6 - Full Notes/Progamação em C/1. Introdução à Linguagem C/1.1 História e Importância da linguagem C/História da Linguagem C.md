# Introdução

A **linguagem C** é considerada uma das linguagens de programação mais influentes da história da computação. Criada na década de 1970, ela serviu de base para diversas outras linguagens modernas (como C++, C#, Java, Python e até mesmo a base de sistemas embarcados).

Sua importância vem do equilíbrio entre **baixo nível** (próximo do hardware) e **alto nível** (facilidade de programação), o que a torna extremamente versátil.

# Antecedentes: A Linguagem B

Antes do Surgimento de C, existiam outras linguagens que inspiraram sua criação:

- **BCPL (*Basic Combined Programming Language*, 1996)**: Criada por Martin Richards, era uma linguagem simples para escrever compiladores e trabalhar com sistemas.
- **B (1969)**: Desenvolvida por **Ken Thompson** no Bell Labs, baseada no BCPL. A linguagem B já era usada no **sistema operacional UNIX**, mas tinha limitações, principalmente em relação ao suporte a tipos de dados.

Essas limitações levaram à evolução natural para uma linguagem mais poderosa: C.

---
# Nascimento de C

- **Criador:** Dennis Ritchie, no Bell Labs (AT&T), em 1972.
- **Objetivo:** criar uma linguagem mais eficiente que B, com **tipagem de dados, operadores mais ricos e controle de baixo nível.**

![[Pasted image 20250819165339.png]]

A primeira grande aplicação da linguagem C foi na **reescrita do sistema operacional UNIX**.

Inicialmente, UNIX, havia sido implementado em assembly, mas foi quase totalmente reescrito em C em 1973.

Isso marcou um grande avanço: o sistema operacional se tornou **portável** entre diferentes máquinas, já que o C podia ser compilado em diversos computadores.

---
# Popularização

Durante os anos 1970 e 1980, C começou a ganhar popularidade por alguns motivos principais:

- **Portabilidade:** programas em C podiam ser compilados em diferentes máquinas sem grandes alterações.
- **Eficiência:** por ser próxima do hardware, permitia controle detalhado da memória e desempenho elevado.
- **Adaptação:** podia ser usada tanto para sistemas operacionais quanto para softwares de aplicação.

O lançamento do livro ***"The C Programming Language"*** (1978), escrito por **Brian Kernighan e Dennis Ritchie**m conhecido como **K&R C**, foi fundamental para disseminar a linguagem e padronizar seu uso.

---
# Padronização

Com o crescimento, surgiram variações de C em diferentes compiladores. Para evitar divergências, começaram os esforços de padronização:

- **ANSI C (1989):**
	O *American National Standards Institute* (ANSI) criou um padrão para a linguagem, também conhecido como **C89**.

- **ISO C (1990):**
	O padrão foi reconhecido pela ISO *(International Organization for Standardization)*, consolidando o **C90**.

- **C99 (1999):**
	Introduziu melhorias como tipos inteiros de tamanho fixo (`int32_t`, `uint64_t`), variáveis declaradas em qualquer lugar do código e novos recursos matemáticos.

- **C11 (2011):**
	Trouxe recursos de concorrência *(threads)* e melhorias de segurança na manipulação de strings.

- **C18 (2018):**
	Pequenas correções no padrão C11.

---
# Legado e Influência

A linguagem C influenciou diretamente muitas linguagens posteriores:

- **C++** (1983) → extensão de C com suporte a orientação a objetos.
- **C#** (2000, Microsoft) → linguagem da plataforma .NET, inspirada em C/C++ e Java.
- **Java, Python, PHP, JavaScript** → possuem sintaxe e conceitos derivados de C.

Além disso, C continua sendo **fundamental** no desenvolvimento de:

- Sistemas Operacionais (Linux, Windows, macOS, Android, iOS)
- Compiladores e interpretadores
- Sistemas embarcados (microcontroladores, IoT, automóveis, aviões, etc.)
- Jogos e aplicações de alta performance

---
# Conclusão

A linguagem C nasceu da necessidade de se criar uma ferramenta poderosa, portátil e eficiente para desenvolvimento de sistemas.  
Mesmo após mais de 50 anos, continua sendo uma das linguagens **mais usadas no mundo**, tanto na indústria quanto na academia.  
Seu impacto é tão grande que praticamente todos os programadores, direta ou indiretamente, aprendem ou usam conceitos que vieram dela.
