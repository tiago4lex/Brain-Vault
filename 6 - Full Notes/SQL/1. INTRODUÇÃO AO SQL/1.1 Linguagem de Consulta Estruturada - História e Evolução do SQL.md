# O que é SQL?

![[Pasted image 20250923154400.png]]

## Definição

SQL *(Structured Query Language)* é uma linguagem de programação específica de domínio projetada para gerenciar dados em sistemas de gerenciamento de banco de dados relacional (RDBMS). É a linguagem padrão para operações em bancos de dados relacionais.

## Características Principais

- **Declarativa:** Você especifica "o quê" quer, não "como" fazer.
- **Padronizada:** Segue padrões ANSI/ISO.
- **Multiplataforma:** Funciona em diversos SGBDs.
- **Completa:** Suporta definição, manipulação e controle de dados.

# História do SQL

## Origens (Anos 70)

- **1970:** Edgar F. Codd publica *"A Relational Model of Data for Large Shared Data Banks"*

![[Pasted image 20250923154515.png]]

- **1974:** Donald D. Chamberlin e Raymond F. Boyce desenvolve a linguagem "SEQUEL" *(Structured English Query Language)*
- **1978:** A IBM  implementa o System R, primeiro protótipo de RDBMS

## Padronização (Anos 80)

- **1986:** Primeiro padrão ANSI SQL
- **1987:** Padrão ISO SQL
- **1989:** SQL-89 (padrão revisado)

## Marcos Importantes

- **1979**: Oracle Corporation lança primeiro RDBMS comercial
- **1983**: IBM lança DB2
- **1985**: Surge o SQL Server da Sybase (posteriormente Microsoft)

# Evolução dos Padrões SQL

## SQL-86/SQL-87

- Primeiro padrão oficial
- Sintaxe básica para consultas
- Operações fundamentais de DML e DDL

## SQL-89 (SQL1)

- Adição de integridade referencial
- Melhorias na sintaxe
- Suporte a chaves primárias e estrangeiras

## SQL-92 - Grande Revisão

```sql
-- Exemplo de novas funcionalidades do SQL-92
SELECT a.nome, b.departamento
FROM funcionarios a
INNER JOIN departamentos b ON a.dept_id = b.id
WHERE a.salario > 50000;
```

**Principais Adições:**

- Junções (JOIN) explícitas.
- Consultas mais complexas.
- Melhor suporte a dados temporais.
- Operações de conjuntos avançados.

## SQL:1999 (SQL3) - Programabilidade

**Principais Inovações:**

- Expressões de tabela comuns (CTEs).
- Tipos de dados definidos pelo usuário.
- Triggers e stored procedures.
- Suporte a objetos.

## SQL:2003 - Modernização

```sql
-- Exemplo: Funções de janela (window functions)
SELECT nome, departamento, salario,
       AVG(salario) OVER (PARTITION BY departamento) as media_dept
FROM funcionarios;
```

**Novas Funcionalidades:**

- Funções de janela (Window functions).
- Tipos de dados XML.
- Sequences.
- Melhorias em OLAP.

## SQL:2006 - Integração XML

- Integração nativa com XML.
- XQuery suportado.
- Publicação de dados relacionais como XML.

## SQL:2011 e SQL:2016 - Era Moderna

**SQL:2011:**

- Dados temporais (temporal tables).
- Padronização de MERGE.
- Melhorias em window functions.

**SQL:2016:**

- JSON suportado.
- Polimorfismo em tabelas.
- Padronização de funcionalidades específicas de vendors.

## SQL:2019 e Além

- Machine Learning no banco de dados.
- Gráficos e consultas recursivas.
- Melhor suporte para análise de dados.

# Arquitetura e Componentes do SQL

## Subconjuntos da Linguagem

| **Componente** | **Sigla**                      | **Função**            | **Exemplos**           |
| -------------- | ------------------------------ | --------------------- | ---------------------- |
| DDL            | *Data Definition Language*     | Definir estrutura     | CREATE, ALTER, DROP    |
| DML            | *Data Manipulation Language*   | Manipular dados       | SELECT, INSERT, UPDATE |
| DCL            | *Data Control Language*        | Controle de acesso    | GRANT, REVOKE          |
| TCL            | *Transaction Control Language* | Controle transicional | COMMIT, ROLLBACK       |

# Diferenças entre Implementações

## Dialetos Principais

```sql
-- Exemplo de diferença entre dialetos

-- MySQL
SELECT * FROM tabela LIMIT 10;

-- SQL Server
SELECT TOP 10 * FROM tabela;

-- Oracle
SELECT * FROM tabela WHERE ROWNUM <= 10;
```

## Comparativo entre SGBDs

| **Características** | **MySQL** | **PostgreSQL** | **SQL Server** | **Oracle** |
| ------------------- | --------- | -------------- | -------------- | ---------- |
| Open Source         | Sim       | Sim            | Não            | Não        |
| Suporte a JSON      | Sim       | Excelente      | Sim            | Sim        |
| Window Functions    | Sim       | Excelente      | Sim            | Sim        |
| CTEs Recursivas     | Sim       | Sim            | Sim            | Sim        |
